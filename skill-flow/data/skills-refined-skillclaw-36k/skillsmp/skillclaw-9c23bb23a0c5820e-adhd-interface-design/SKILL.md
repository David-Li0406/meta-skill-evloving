# Skill body

## Purpose

Design interfaces that function as **cognitive prosthetics** — external systems that actively compensate for executive dysfunction. ADHD is a regulation deficit, not an attention deficit.

## Core Principles

1. **Minimal Friction**: Reduce cognitive load by minimizing the number of actions required to complete tasks.
2. **Clear Single Priority**: Present one task at a time to avoid overwhelming the user.
3. **Batched Delivery**: Consolidate updates and deliver information at intervals to reduce context-switching costs.
4. **Context Always Included**: Automatically surface relevant information to support working memory.
5. **Reduced Decision Fatigue**: Simplify decision-making by providing clear recommendations and pre-loaded actions.

## Executive Functions to Externalize

| Function         | Implementation                                           |
|------------------|---------------------------------------------------------|
| **Initiation**   | Pre-load the next action; use micro-commitments (e.g., "just 2 minutes"). |
| **Prioritization** | Use AI to rank tasks and provide one clear recommendation with reasoning. |
| **Monitoring**   | Implement visual timers, progress bars, and task boundary detection. |
| **Memory**       | Include an auto-populated sidebar with essential information (What/Why/Next/When/Source) for each task. |
