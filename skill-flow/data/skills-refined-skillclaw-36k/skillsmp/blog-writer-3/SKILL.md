# Blog Writer

Convert Showroom Asciidoc labs and demos into engaging technical blog articles.

## Usage

Provide the path to a Showroom lab or demo content to transform into a blog post.

## Transformation Guidelines

1. **Extract Key Concepts**: Identify the main learning objectives and technical concepts
2. **Narrative Flow**: Transform step-by-step instructions into a narrative format
3. **Code Highlights**: Select and present the most important code snippets
4. **Visual Elements**: Suggest diagrams or screenshots where appropriate
5. **Call to Action**: Include links to the full lab for hands-on practice

## Output Format

- Markdown format suitable for publishing
- SEO-friendly title and meta description
- Appropriate headers and sections
- Engaging introduction and conclusion
