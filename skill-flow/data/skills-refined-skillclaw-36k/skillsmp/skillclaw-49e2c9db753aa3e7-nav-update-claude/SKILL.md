---
name: nav-update-claude
description: Use this skill when you need to update the project CLAUDE.md to the latest Navigator version while preserving customizations.
---

# Skill body

1. **Identify the Current Version**: Check the current version of the Navigator configuration in CLAUDE.md.
2. **Check for Updates**: Look for the latest version of the Navigator.
3. **Backup Customizations**: Ensure that any customizations in CLAUDE.md are backed up.
4. **Update CLAUDE.md**: Replace the outdated Navigator configuration with the latest version.
5. **Restore Customizations**: Reapply any customizations that were backed up.
6. **Test the Update**: Verify that the updated CLAUDE.md works as expected with the new Navigator version.
7. **Document Changes**: Record the changes made during the update process for future reference.
