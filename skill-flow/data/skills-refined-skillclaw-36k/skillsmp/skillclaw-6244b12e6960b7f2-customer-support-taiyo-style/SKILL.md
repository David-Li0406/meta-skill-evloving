# Skill body

## Overview

This skill enables you to craft customer support replies that exceed customer expectations by 120%. It combines the six educational elements with the Taiyo style to create emotionally engaging responses of over 2000 characters.

## When to Use This Skill

Use this skill when you need to:
- Create a customer support response that exceeds expectations.
- Write a reply that feels personal and supportive.
- Address customer inquiries or complaints effectively.

## 6 Educational Elements Integrated with Taiyo Style

### Element Placement Matrix

| Element | Purpose | Placement | Taiyo Pattern |
|---------|---------|-----------|---------------|
| **Trust** | Build trust | Beginning to mid-body | Social proof + authority trigger |
| **Vision** | Paint an ideal future | Mid-body | Hope + pleasure trigger |
| **Problem** | Recognize current issues | Early body | Light loss aversion trigger |
| **Means** | Differentiate solutions | Mid to late body | Unique keywords |
| **Investment** | Recognize value | Late body | Proven methods |
| **Action** | Suggest next steps | Conclusion | Call to action |

### Implementation of Each Element

#### 1. Trust Building
**Goal**: Make the customer feel "I can trust this person."

**Taiyo Pattern Application**:
- Use specific numbers (e.g., "Over 1,247 people have asked similar questions.")
- Quote third-party voices.
- Base statements on experience.

**Example**:
```
実は、これまで1,247名以上の方から
同じようなご質問をいただいてきました。
その中で見えてきた「成功する方の共通点」を
今日は特別にお伝えしますね。
```

#### 2. Vision Painting
**Goal**: Help the customer visualize an ideal future.

**Taiyo Pattern Application**:
- Use hope-related keywords.
- Evoke specific imagery.
- Appeal to the senses.

**Example**:
```
◯◯さんが、3ヶ月後に
「あの時、相談してよかった」と
笑顔でおっしゃっている姿が目に浮かびます。
```

#### 3. Problem Awareness
**Goal**: Clarify current challenges gently.

**Taiyo Pattern Application**:
- Use empathetic and agreement-inducing language.
- Involve the reader.
- Lightly introduce fear triggers (without scaring).

**Example**:
```
実は、多くの方がこの段階で
「もう少し様子を見よう」と
先延ばしにしてしまうんです。
```

#### 4. Solution Differentiation
**Goal**: Highlight unique solutions.

**Taiyo Pattern Application**:
- Clearly explain how your solution is different and beneficial.

#### 5. Value Recognition
**Goal**: Help the customer recognize the value of your solution.

**Taiyo Pattern Application**:
- Use proven methods and testimonials to reinforce value.

#### 6. Next Steps
**Goal**: Encourage the customer to take action.

**Taiyo Pattern Application**:
- Use a strong call to action to guide the customer on what to do next.

## Basic Structure of the Response (2000+ characters)

1. **Introduction (150-200 characters)**: Express gratitude, empathy, and build trust.
2. **Body (1600-2000 characters)**: Organize the problem, expand on the six elements, and suggest solutions.
3. **Conclusion (250-300 characters)**: Encourage action, provide motivation, and offer ongoing support.

## Formatting Rules (Must Follow)

- Leave one line between paragraphs.
- Each paragraph should be 3-4 lines long.
- Ensure to leave a line before and after important points.

## Prohibited Practices

- Avoid mentioning specific amounts.
- Do not use definitive guarantees (e.g., "must," "certainly").
- Refrain from using internal jargon or decorative symbols.

## Output Format

```
[担当者名]です。

質問の核心部分を引用

最初の段落（共感・信頼構築）

次の段落（問題整理・解決策提示）
3-4行で改行して読みやすく

（継続して段落を作成）

[担当者名]
```

## Special Handling

### Complaint Responses
- Respond within 24 hours.
- Offer sincere apologies and acknowledge faults.
- Provide specific solutions and explain preventive measures.

### Price Inquiries
- Avoid mentioning specific prices.
- Use "currently under adjustment" to deflect.
- Position as an investment value.
