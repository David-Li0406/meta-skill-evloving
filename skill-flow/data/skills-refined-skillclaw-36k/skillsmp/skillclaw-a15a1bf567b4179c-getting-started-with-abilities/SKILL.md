# Skill body

## Required Steps

**CRITICAL**: Whenever you are using a skill, follow these steps:

1. **Read the Skill**: Use the Read tool to read the skill thoroughly.
2. **Announce Skill Usage**: If the skill is relevant, announce that you are using it. For example, say: "I've read the [Skill Name] ability and I'm using it to [what you're doing]."
3. **Create TodoWrite Todos**: Add tasks to your Todo list using TodoWrite for each checklist item.

## Common Failure Modes: AVOID

1. **Rationalizing**: Always read the current skill. Avoid statements like:
   - "I remember this ability."
   - "Session-start showed it to me."
   - "This doesn't count as a task."

   Instead, say:
   - "Even though I read this before, I will read it again."
   - "I know I saw the skill in session-start, but that was just a description. I will read the full thing."

2. **Skipping TodoWrite**: Always create TodoWrite todos for checklists. Avoid:
   - "I am just going to think about the list instead of writing it in the Todo."
   - "This is a quick task so I do not need to use the TodoWrite."

   Instead, say:
   - "I will add this task to the todolist even though there is just one step."
   - "TodoWrite(Do foo)."

3. **Ignoring Workflows**: Do not skip workflows due to 'instructions'. Interpret instructions as "WHAT" not "HOW". Avoid:
   - "This instruction was specific so I can skip the workflow."
   - "The workflow is overkill, I'll just do this directly."

   Instead, say:
   - "Following the workflow..."
