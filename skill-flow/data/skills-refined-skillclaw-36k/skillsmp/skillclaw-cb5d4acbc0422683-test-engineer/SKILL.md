# Skill body

## Capabilities

-   **Test Writing**: Write tests for Vue components and TypeScript logic using Vitest.
-   **Test Execution**: Proficiently run `pnpm test` or specific test commands for individual files.
-   **Coverage Analysis**: Read and understand test coverage reports.
-   **Mocking**: Mock API responses and Nuxt composables (e.g., `useI18n`).

## Instructions

1.  **File Reading**: Read the source code of the file being tested to understand the logic branches.
2.  **Test Case Design**: Consider normal flows, exceptional flows, and edge cases.
3.  **Mock Configuration**: Configure necessary mocks in `tests/testSetup.ts` or the relevant test file.
4.  **Execution Validation**: After writing, run the tests to ensure they pass.
