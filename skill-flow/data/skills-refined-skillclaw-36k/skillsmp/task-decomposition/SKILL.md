# 任务分解与规划技能

## 概述

该技能帮助用户将复杂任务分解为可执行的子任务，并制定详细的执行计划。适用于项目规划、时间管理、任务优化等场景。

## 核心功能

### 1. 任务分解
- 将大型任务拆分为可管理的子任务
- 识别任务的关键组成部分
- 确定任务复杂度和难度级别

### 2. 顺序规划
- 分析任务依赖关系
- 确定任务执行的先后顺序
- 创建任务执行路径图

### 3. 时间表制定
- 估算每个任务的所需时间
- 制定详细的执行时间表
- 设置合理的里程碑和截止日期

### 4. 资源评估
- 评估任务所需的资源（人力、物力、财力）
- 识别潜在的资源瓶颈
- 优化资源分配方案

### 5. 流程优化
- 识别任务流程中的低效环节
- 提出改进建议
- 优化任务执行流程

## 使用方法

### 基本流程
1. 用户描述需要处理的事情
2. 技能自动激活，分析任务复杂度
3. 分解任务为子任务并确定顺序
4. 制定详细的执行计划和时间表
5. 提供优化建议和资源评估

### 输入格式
- 清晰描述需要处理的任务或项目
- 包含任务的目标、范围和约束条件
- 提供可用资源和时间限制信息

### 输出格式
- 任务分解树形图
- 执行顺序时间表
- 资源需求评估
- 优化建议和风险提示

## 技术支持

该技能结合了项目管理方法论和优化算法，确保任务分解的科学性和实用性。
