# Skill body

## Overview

The Foundation Core skill encompasses essential principles and architectural patterns that ensure quality, efficiency, and scalability in AI-powered development. It is designed to guide the creation and orchestration of workflows using proven methodologies.

## Core Principles

1. **TRUST 5 Framework**: A quality gate system that emphasizes Test-first, Readable, Unified, Secured, and Trackable development.
2. **SPEC-First DDD**: A specification-driven domain-driven development workflow that prioritizes clear specifications.
3. **Delegation Patterns**: Orchestrate tasks through specialized agents rather than direct execution, enhancing modularity and flexibility.
4. **Token Optimization**: Manage a budget of 200K tokens efficiently to ensure context-aware operations.
5. **Progressive Disclosure**: Implement a three-tier knowledge delivery system (Quick, Implementation, Advanced) to facilitate learning and application.
6. **Modular System**: Utilize file splitting and reference architecture to promote scalability and maintainability.
