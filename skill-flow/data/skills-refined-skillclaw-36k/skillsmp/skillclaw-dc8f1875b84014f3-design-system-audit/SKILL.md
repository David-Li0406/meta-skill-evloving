# Skill body

## 1. 여백 및 그리드 시스템 (Spacing)

- **컴포넌트 간격**: 모든 UI 컴포넌트 간의 간격이 일관되게 유지되는지 확인합니다.
- **내부 여백**: 각 컴포넌트의 내부 여백이 디자인 시스템에 맞는지 검토합니다.

## 2. 스타일 통일성 (Consistency)

- 색상, 타이포그래피, 아이콘 등의 스타일이 일관되게 적용되었는지 확인합니다.

## 3. 검토 절차
1. UI 관련 코드가 수정되거나 생성될 때, 디자인 시스템 규칙이 적용되었는지 확인합니다.
2. 컴포넌트의 가독성과 시각적 계층 구조가 명확한지 평가합니다.

## 실행 가이드
1. 사용자가 "디자인리뷰 해줘"라고 요청하면 이 지침을 따릅니다.
