# Theme Brushes

## Brush keys

Keep theme dictionaries limited to colors and brushes:

```
Brushes/Surface
Brushes/SurfaceAlt
Brushes/Border
Brushes/TextPrimary
Brushes/TextSecondary
Brushes/Accent
```

## Notes

- Theme dictionaries should not define control templates.
- Use the same keys across Light/Dark for easy swapping.
