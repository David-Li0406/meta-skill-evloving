name: jimeng-api
description: Use this skill when users request AI-generated images from the Jimeng (即梦AI) service, including artwork, illustrations, or visual content creation. It supports both text-to-image and image-to-image generation with customizable ratios and resolutions.
---

# Skill body

1. **Input Requirements**:
   - Provide a text prompt for image generation.
   - Specify any desired customization options (e.g., ratios, resolutions).

2. **Image Generation**:
   - Call the Jimeng API with the provided text prompt and customization options.
   - For image-to-image generation, include the source image along with the prompt.

3. **Output**:
   - Receive the generated image from the API.
   - Display or save the image as needed.
