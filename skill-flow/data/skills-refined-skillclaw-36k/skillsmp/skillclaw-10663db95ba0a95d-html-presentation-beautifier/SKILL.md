# HTML Presentation Beautifier

Transform documents and data into professional McKinsey-style HTML presentations with intelligent content structuring, visualizations, and interactive navigation.

## Core Principles

- **Content Preservation**: Maintain 100% of the original content while applying a professional design.
- **Audience Awareness**: Tailor the presentation complexity and formality based on the audience.
- **Clear Messaging**: Focus on the single most important takeaway for the audience.
- **Visual Hierarchy**: Emphasize key data and insights effectively.

## When to Use This Skill

Trigger scenarios:
- "Create a presentation from this document/report"
- "Transform this analysis into slides"
- "Generate McKinsey-style presentation"
- "Visualize this data professionally"
- "Make HTML slides from this content"

## Process Overview

This skill follows a structured multi-phase process:

1. **Phase 1: Document Parsing** (Approx. 1 min)
   - Read the source document completely.
   - Identify document type (report, analysis, research).
   - Extract structural elements: headings, bullet points, data tables, key conclusions.

2. **Phase 2: Plan Slides** (Approx. 2 min)
   - Organize content into a slide-friendly format.
   - Create a slide plan that includes visualizations.

3. **Phase 3: Apply Design** (Approx. 1 min)
   - Implement McKinsey-style design principles, ensuring a clean and professional look.

4. **Phase 4: Content Visualization** (Approx. 2 min)
   - Enhance the presentation with intelligent chart selection and graphics.

5. **Phase 5: Generate HTML** (Approx. 3 min)
   - Produce a single-file HTML presentation with interactive features.

6. **Phase 6: Review & Verify** (Approx. 2 min)
   - Conduct a quality review and generate a report with scores.

**Total Time**: Approximately 10-12 minutes for a typical presentation.

## Design System (McKinsey/BCG Style)

### Color Palette

| Color Type | Hex Code | Usage |
|------------|----------|-------|
| Primary Background | `#FFFFFF` | Slide background |
| Header Background | `#000000` | Title bars |
| Primary Accent | `#F85d42` | Key highlights, CTAs |
| Secondary Accent | `#74788d` | Supporting text |
| Deep Blue | `#556EE6` | Charts, data points |
| Green | `#34c38f` | Success indicators |
| Blue | `#50a5f1` | Neutral emphasis |
| Yellow | `#f1b44c` | Warnings, notes |

### Typography

- **Title**: Large (48-64px), Bold, Black (`#000000`)
- **Subtitle**: Medium (28-36px), Bold, Accent Color
- **Body Text**: Regular (16-20px), Dark Gray (`#333333`)
- **Emphasis**: Regular with Accent Color or Bold
- **Chart Labels**: Small (12-14px), Clear, Readable

### Design Principles

- **Modern Business Style**: Clean, professional, minimal clutter.
- **Visual Hierarchy**: Clear distinction between titles, subtitles, and body.
- **Consistency**: Uniform design language across all slides.
- **Data-Driven**: Use charts and visualizations for quantitative information.
- **Professional Polish**: Ensure high-quality presentation standards.
