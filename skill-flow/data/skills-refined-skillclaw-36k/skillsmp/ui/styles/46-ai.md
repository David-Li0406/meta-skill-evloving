# 46. AI 風格 AI/ML

## 概述
展現人工智慧與機器學習科技感的設計風格。

## 特徵
- 神經網路視覺
- 數據流動感
- 智能氛圍
- 未來科技

## 配色
```css
:root {
  --ai-dark: #0a0a1a;
  --ai-primary: #6366f1;
  --ai-secondary: #8b5cf6;
  --ai-accent: #06b6d4;
  --ai-glow: rgba(99, 102, 241, 0.5);
}
```

## 元件

### 神經網路背景
```css
.ai-network {
  background: var(--ai-dark);
  position: relative;
  overflow: hidden;
}

.ai-network::before {
  content: '';
  position: absolute;
  inset: 0;
  background: 
    radial-gradient(circle at 20% 30%, var(--ai-glow) 0%, transparent 50%),
    radial-gradient(circle at 80% 70%, rgba(139, 92, 246, 0.3) 0%, transparent 50%);
}
```

### 數據粒子
```css
.ai-particle {
  width: 4px;
  height: 4px;
  background: var(--ai-accent);
  border-radius: 50%;
  box-shadow: 0 0 10px var(--ai-accent);
  animation: pulse 2s infinite;
}

@keyframes pulse {
  0%, 100% { opacity: 0.5; transform: scale(1); }
  50% { opacity: 1; transform: scale(1.5); }
}
```

### 漸層文字
```css
.ai-gradient-text {
  background: linear-gradient(135deg, var(--ai-primary), var(--ai-secondary), var(--ai-accent));
  -webkit-background-clip: text;
  -webkit-text-fill-color: transparent;
}
```

### 卡片
