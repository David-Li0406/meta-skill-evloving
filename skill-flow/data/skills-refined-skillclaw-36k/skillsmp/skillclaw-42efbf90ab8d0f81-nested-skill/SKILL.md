# Nested Skill

This skill demonstrates how to organize skills in a nested directory structure.

**Location**: `category-a/subcategory-1/SKILL.md` (example for depth 3)
**Depth**: Up to 5 levels from root

Skills can be organized in categories and subcategories to enhance organization and facilitate discovery. You can structure your skills in any nested format that suits your needs.
