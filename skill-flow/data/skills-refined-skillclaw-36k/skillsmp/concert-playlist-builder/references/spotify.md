# Spotify Web Player

## Setup

1. Navigate to `https://open.spotify.com`
2. User must be signed in (prompt if not)
3. Get tab context with `tabs_context_mcp`

## Create New Playlist

1. In left sidebar, click "Create playlist" or "+" button
2. New playlist appears in sidebar (named "My Playlist #N")
3. Click to open, then click playlist name to rename
4. Enter event name and confirm

## Add Songs to Playlist

### Search Pattern

```
1. Click search box (top area, or use Ctrl/Cmd+K)
2. Type artist name
3. Wait for results (instant search, no Enter needed)
4. Results show Artists, Songs, Albums sections
```

### Add Song Pattern

```
1. In search results, hover over a song row
2. Click "..." button (three dots, appears on right)
3. Click "Add to playlist"
4. Select target playlist from submenu
```

### Alternative: Drag and Drop

```
1. Search for artist
2. Drag song from results to playlist in left sidebar
```

### Add from Artist Page

```
1. Click artist name to go to artist page
2. Find "Popular" tracks section
3. Hover over track, click "..." → "Add to playlist"
```

### Element Patterns

- Search: Top bar, usually has keyboard shortcut icon
- Song "..." button: Right side of track row, appears on hover
- "Add to playlist" submenu: Shows recent playlists first

### Tips

- Spotify search is instant - no need to press Enter
- Right-click on tracks also opens context menu
- Can add entire albums: "..." → "Add to playlist"
- Spotify has most artists but some underground/regional may be missing
- Green "+" button adds to Liked Songs, use "..." for playlists
