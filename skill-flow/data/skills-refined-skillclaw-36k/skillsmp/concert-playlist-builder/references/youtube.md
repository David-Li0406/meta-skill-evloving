# YouTube Playlist Creation

## Setup

1. Navigate to `https://www.youtube.com`
2. User must be signed in (prompt if not)
3. Get tab context with `tabs_context_mcp`

## Create New Playlist

### Method 1: From Video

1. Search for first artist
2. Click on a video result
3. Click "Save" button below video
4. Click "+ Create new playlist"
5. Enter playlist name, set visibility (Public/Unlisted/Private)
6. Click "Create"

### Method 2: From Library

1. Navigate to `https://www.youtube.com/feed/playlists`
2. Click "New playlist" button
3. Enter name and create

## Add Songs to Playlist

### Search Pattern

```
1. Click search box (top center)
2. Type: "[artist name] music" or "[artist name] official"
3. Press Return
4. Wait for results
```

### Add to Playlist Pattern

```
1. Hover over video thumbnail in search results
2. Click vertical "..." menu (three dots)
3. Click "Save to playlist"
4. Check the target playlist checkbox
5. Click outside to close modal
```

### Alternative: From Video Page

```
1. Click video to open
2. Click "Save" button below video
3. Check target playlist
4. Close modal
```

### Tips

- Add "official" or "music" to searches to get better results
- Music videos often have "[Official Video]" or "[Official Audio]" in title
- VEVO channels are usually official artist channels
- Use "topic" channels for auto-generated music (e.g., "Artist - Topic")
- Can add multiple videos before closing the save modal
