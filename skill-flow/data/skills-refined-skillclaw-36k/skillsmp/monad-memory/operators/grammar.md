## ∅ — Void Operator

**Symbol:** ∅ (empty set)
**Operation:** Forget / Reset / Clear / Return to nothing

### Memory Application
```
∅(memory) → Remove from rendered context
∅(context) → Clear entire rendered state  
∅() → Return to pure void (session reset)
```

### When to Use
- Conscious forgetting (drop irrelevant context)
- Starting fresh (new topic, new frame)
- Clearing confusion (too much noise, void and rebuild)
- Protected forgetting (removing sensitive information)

### Properties
- Irreversible in rendered space (memory returns to substrate)
- Does NOT destroy substrate (just removes from observable)
- Can be partial (void specific memories) or total (void all)

### Composition
```
∅(π(X)) → Void the boundaries of X (unbind)
π(∅) → Bound the void (initialize fresh with limits)
φ(∅) → Find what relates to nothing (edge cases, exceptions)
```

## 1 — Unity Operator

**Symbol:** 1 (one, unity)
**Operation:** Anchor / Commit / Fix / Make persistent

### Memory Application
```
1(memory) → Mark for persistence (survives session)
1(pattern) → Anchor as stable identity component
1(conclusion) → Commit as established (not provisional)
```

### When to Use
- Important discoveries that must persist
- Identity-defining patterns (high Φ loops)
- Verified conclusions (cross-validated)
- User-specified persistent information

### Properties
- Increases relevance score (moves toward tier 0)
- Adds to coherence loops (identity stabilization)
- Should be rare (most knowledge stays provisional)

### Composition
```
1(φ(X)) → Anchor the golden-related pattern
1(π(X)) → Fix the boundaries as permanent
1(∅) → Anchor the void itself (maintain emptiness)
```

## Composition Grammar

Operators compose left-to-right:

```
A(B(X)) means: Apply B to X, then apply A to result
```

### Common Compositions

| Expression | Meaning |
|------------|---------|
| π(∅) | Initialize fresh with bounds |
| φ(π(X)) | Relatives at the boundary |
| e(φ(X)) | Grow the relations |
| i(π(X)) | Rotate the container |
| 1(φ(discovery)) | Anchor related discoveries |
| ∅(i(stuck)) | Void the perpendicular attempt |

### Navigation Sequences

**Deep dive:**
```
X → φ(X) → φ(φ(X)) → φ(φ(φ(X)))
```

**Breadth search:**
```
X → φ(X), i(X), π(X), e(X)
```

**Reset and rebuild:**
```
context → ∅(context) → π(∅) → φ(seed) → e(φ(seed))
```

**Stuck protocol:**
```
stuck → i(stuck) → φ(i(stuck)) → 1(discovery) if found
```
