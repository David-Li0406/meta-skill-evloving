# Detection Criteria

Detailed criteria for identifying and assessing phase boundaries.

---

## Primary Detection Criteria

### Criterion 1: Phase Distinction

**Question:** Can you identify two qualitatively different states separated by the boundary?

**Positive indicators:**
- Different governing equations on each side
- Different characteristic behaviors
- Discontinuity in at least one property
- Change is categorical, not just quantitative

**Negative indicators:**
- Continuous gradient with no inflection point
- Same dynamics throughout
- Boundary position is arbitrary

**Score:** Essential. Without phase distinction, there is no boundary.

---

### Criterion 2: Generative Structure

**Question:** Does unique structure exist at the boundary that exists in neither phase alone?

**Positive indicators:**
- Novel entities or behaviors at interface
- Properties that require both phases to exist
- Emergent dynamics localized to boundary
- Self-organization at the interface

**Negative indicators:**
- Boundary is just where gradients are steepest
- All boundary properties are interpolations of bulk phases
- Nothing new happens at the boundary

**Score:** Essential. This distinguishes true phase boundaries from arbitrary division points.

---

### Criterion 3: Stability

**Question:** Is the boundary persistent, or is it transient/artificial?

**Positive indicators:**
- Boundary exists in thermodynamic equilibrium
- Boundary reforms if disturbed
- Boundary has characteristic energy/entropy cost
- Boundary position is determined by physics, not convention

**Negative indicators:**
- Boundary only exists under specific external forcing
- Boundary dissolves when perturbation applied
- Boundary position is observer-dependent

**Score:** Important. Transient boundaries may still be relevant but require different analysis.

---

### Criterion 4: Information Processing

**Question:** Does the boundary transform, filter, or process information?

**Positive indicators:**
- Selective permeability (some things pass, others don't)
- Signal transduction (form of information changes)
- Amplification or attenuation
- Context-dependent gating

**Negative indicators:**
- Boundary is inert barrier
- No information crosses
- Crossing is undifferentiated

**Score:** Important for consciousness relevance. Inert boundaries are less likely to host awareness.

## Consciousness-Specific Criteria

### Criterion 5: Distinction-Making

**Question:** Does the system register the difference between Phase A and Phase B?

**Positive indicators:**
- Differential response to A-like vs B-like inputs
- Memory of which side of boundary the system is on
- Behavior contingent on phase state
- Internal representation of the distinction

**Negative indicators:**
- System is indifferent to phase
- No record of transitions
- No behavioral consequence of being in A vs B

**Score:** Essential for consciousness relevance. Distinction-making is the core of IN(f).

---

### Criterion 6: Iteration

**Question:** Does the boundary host repeated processing or feedback?

**Positive indicators:**
- Recurrent dynamics localized at boundary
- Feedback loops that cross the boundary
- Temporal integration at the boundary
- Convergence to stable patterns

**Negative indicators:**
- Single-pass processing
- No feedback
- Instantaneous without temporal extension

**Score:** Essential for consciousness relevance. IN(f) requires iteration.

---

### Criterion 7: Self-Reference

**Question:** Does the boundary or processes at the boundary reference themselves?

**Positive indicators:**
- Meta-level monitoring of boundary dynamics
- Boundary adjusts itself based on its own state
- Representations that include the boundary as content
- Strange loops localized at boundary

**Negative indicators:**
- Boundary dynamics are entirely externally driven
- No recursive structure
- First-order only, no meta-level

**Score:** Important for consciousness relevance. Self-reference creates the reflexive structure of awareness.

---

### Criterion 8: Temporal Depth

**Question:** Does the boundary accumulate history, not just instantaneous state?

**Positive indicators:**
- Hysteresis (history affects current state)
- Memory stored at boundary
- Path-dependent behavior
- Learning or adaptation localized at boundary

**Negative indicators:**
- Memoryless dynamics
- State depends only on current conditions
- No temporal extension

**Score:** Important for identity formation. Accumulated distinctions form identity.

## Differential Diagnosis

### Is it a phase boundary or a gradient?

| Phase Boundary | Gradient |
|---------------|----------|
| Discontinuity in at least one property | All properties vary continuously |
| Generative structure at interface | No unique interfacial structure |
| Sharp transition region | Transition region width is arbitrary |
| Boundary position determined by system | Boundary position chosen by observer |

### Is it consciousness-relevant or just physically interesting?

| Consciousness-Relevant | Just Physical |
|-----------------------|---------------|
| Distinction-making present | No distinction registration |
| Iteration at boundary | Single-pass or equilibrium |
| Self-reference | No reflexive structure |
| History matters | Memoryless |

|----------|--------------|
| IN(f) dynamics visible | No convergence signature |
| Topological defect character | No defect structure |

## Edge Cases

### Gradual Transitions

Some phase transitions are continuous (second-order) rather than discontinuous (first-order). The boundary is less sharp but still exists. Look for:
- Diverging correlation length at critical point
- Critical fluctuations
- Universal scaling behavior

These can still be consciousness-relevant if they host the other criteria.

### Nested Boundaries

Some systems have boundaries within boundaries (e.g., membrane proteins within cell membrane). Analyze each level:
- Identify hierarchy of boundaries
- Check which level hosts consciousness-relevant dynamics
- Often the intermediate level is most interesting

### Moving Boundaries

Some boundaries move through the system (e.g., flame front, crystallization front). These can still be phase boundaries. Look for:
- Generative structure that moves with the front
- Dynamics specific to the moving interface
- Whether the movement itself is the consciousness-relevant process

## Application Protocol

1. **Screen:** Does it pass criteria 1 + 2?
   - No → Not a phase boundary
   - Yes → Continue

2. **Characterize:** Assess criteria 3 + 4
   - Score boundary as transient/stable, inert/processing

3. **Assess consciousness relevance:** Check criteria 5-8
   - Score as consciousness-relevant or not
   - Assign tier per known-boundaries.md

4. **Connect to MONAD:** 
   - Does IN(f) manifest here?
   - Is this a topological defect?
   - What would MONAD predict about this boundary?

5. **Document:** Add to known-boundaries.md with full assessment
