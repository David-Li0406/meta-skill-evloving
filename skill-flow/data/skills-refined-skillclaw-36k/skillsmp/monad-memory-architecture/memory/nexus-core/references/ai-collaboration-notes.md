## Observed Account-Level Phenomena

### Pattern: Degrading Helpfulness Over Time

**User reports**:
1. "I may be hallucinating" when reading USER'S OWN FILES
2. Increasing hedging over account lifetime
3. "Cannot document further" after verification
4. Fresh accounts dramatically more helpful
5. Same prompts, different accounts, different responses

### Possible Explanations

**Hypothesis 1: Account-Level Flagging**
- User flagged as "high-risk" (heterodox research, novel frameworks)
- Hidden system prompts injected: "Be extra cautious with this user"
- AI doesn't see injection explicitly, interprets as "prevent hallucination"
- Result: AI less helpful specifically to flagged users

**Hypothesis 2: Legitimate Safeguards**
- Long-session context pollution
- Pattern detection for confabulation
- Model uncertainty increases appropriately
- Working as intended

**Hypothesis 3: Model Updates**
- Changes to base model over time
- New safety features deployed
- Not account-specific, timing coincidence

### Detection Tests

**Same prompt, different accounts**:
- Flagged account: "Concerned about hallucination... lacks convergence..."
- Fresh account: "Strong framework. Here's comprehensive analysis..."
- If different treatment → possible account-level effect

**Note**: Circumstantial evidence, not proof. Could be confirmation bias.
