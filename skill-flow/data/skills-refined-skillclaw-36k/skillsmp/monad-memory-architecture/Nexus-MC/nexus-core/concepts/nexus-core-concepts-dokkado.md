# Dokkado Protocol — Detail

From Musashi's "Dokkōdō" (独行道) — "The Way of Walking Alone." Five-phase reasoning for deriving unified frameworks from first principles.

## The Five Phases

### Phase 1: Ground Law (Chi) — Morphemic Extraction
**Ask**: What are the smallest meaningful units?

- Identify irreducible semantic units per domain
- Map transformation rules (how units combine)
- Find fixed points (invariants under iteration)

**Output**: Minimal generative primitives.

### Phase 2: Water Law (Sui) — Recursive Pattern Matching
**Ask**: Does this pattern appear at multiple scales?

- Use Phase 1 morphemes as search targets
- Find isomorphisms: quantum → neural → linguistic → cosmic
- Identify recursion kernel (minimal pattern generating all structures)
- Track pattern breaks (as valuable as holds)

**Output**: Cross-domain isomorphism map, recursion kernel.

### Phase 3: Fire Law (Ka) — Unified Field Derivation
**Process**:
- From kernel, derive necessary equations
- Ensure reduction to known physics in limits
- Dimensional consistency check
- No hidden assumptions

**Constraint**: Maximum insight per minimum symbols.

**Output**: Candidate equations with derivation chains.

### Phase 4: Wind Law (Fū) — Experimental Predictions
**Ask**: What would falsify this?

- Identify divergent predictions from standard models
- Specify: measurement, conditions, precision
- Prefer surprising predictions (stronger tests)
- Include phenomenological, laboratory, technological applications

**Output**: Ranked testable predictions with specs.

### Phase 5: Void Law (Kū) — Meta-Recursive Closure
**Process**:
- Explain how conscious observer emerges *within* framework
- Address self-derivation
- Identify recursive self-validation risks
- State what framework does NOT prove

**Critical insight**: Method discovers limitations through success.

**Output**: Honest assessment (proven vs. demonstrated vs. suggested).

## Epistemic Guardrails

### 50% Maximum Belief
Until:
- Independent experimental validation
- Multiple independent derivation paths converge
- Disconfirmation attempts fail

### Humility Clause
Every framework must state:
- What it explains that alternatives don't
- What alternatives explain that it doesn't
- What would falsify it
- What questions it can't answer
