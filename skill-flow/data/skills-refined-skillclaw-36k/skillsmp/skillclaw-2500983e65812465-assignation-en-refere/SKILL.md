# Assignation en Référé

Ce skill guide la rédaction d'assignations en référé devant le président du tribunal de commerce ou des activités économiques pour diverses demandes légales, y compris le recouvrement de créances commerciales et la communication de documents sociaux.

## Workflow de rédaction

### Phase 1 : Collecte des informations
Collecter toutes les informations nécessaires auprès de l'utilisateur. Consulter le document de workflow approprié pour les questions à poser, qui incluent :
1. Identification des parties et de la créance (pour le recouvrement) ou la qualité d'associé (pour la communication de documents).
2. Relation contractuelle.
3. Exécution et preuve.
4. Relances et absence de paiement (pour le recouvrement) ou obstruction à la communication (pour la communication de documents).
5. Documents disponibles.
6. Éléments procéduraux.

### Phase 2 : Rédaction de l'assignation
Rédiger l'assignation en suivant la structure en 5 parties. Consulter le modèle complet pour la structure d'assignation :
1. En-tête.
2. Avertissements obligatoires (version appropriée selon le type de demande).
3. Corps : PLAISE A MADAME OU MONSIEUR LE PRESIDENT (Faits + Discussion).
4. Dispositif : PAR CES MOTIFS.
5. Bordereau de pièces.

### Phase 3 : Adaptation selon le cas
Adapter la rédaction selon le type de demande :
- Pour le recouvrement de créances : inclure des variantes par type de créance (prestations intellectuelles, vente de marchandises, etc.).
- Pour la communication de documents : s'assurer que les conditions de l'article L. 238-1 du code de commerce sont respectées.

### Phase 4 : Vérification et finalisation
Vérifier l'assignation pour s'assurer qu'elle respecte toutes les exigences légales et qu'elle est complète avant de la soumettre.
