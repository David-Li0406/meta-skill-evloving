# URL State Management with nuqs

To sync React state to URL query parameters, follow these steps:

## Prerequisites

Complete these recipes first (in order):

1. **Next.js on Vercel**  
   Create a Next.js app running on Bun, configure the development environment, and deploy to Vercel with automatic deployments on push.

   ```bash
   curl -H "Accept: text/markdown" https://fullstackrecipes.com/api/recipes/nextjs-on-vercel
   ```

## Steps to Implement URL State Management

1. **Set Up nuqs**  
   Sync React state to URL query parameters for shareable filters, search queries, and deep links to modal dialogs. This setup preserves UI state on browser back/forward navigation.

   ```bash
   curl -H "Accept: text/plain" https://fullstackrecipes.com/api/recipes/nuqs-setup
   ```

2. **Manage State with nuqs**  
   Learn to manage React state in URL query parameters, including handling Suspense boundaries, parsers, clearing state, and creating deep-linkable dialogs.

   ```bash
   curl -H "Accept: text/markdown" https://fullstackrecipes.com/api/recipes/using-nuqs
   ```
