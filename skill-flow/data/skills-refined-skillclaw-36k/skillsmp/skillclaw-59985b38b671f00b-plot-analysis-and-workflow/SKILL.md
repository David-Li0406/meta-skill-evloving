# Skill body

## 功能

深入分析故事情节并协调生成大情节点与详细情节点的工作流。

## 使用场景

- 细化和扩展故事大纲。
- 为剧本写作提供详细的情节指导。
- 一键生成故事的大情节点和详细情节点。
- 需要模块化智能体协作的复杂分析任务。

## 核心步骤

1. **理解核心要素**: 仔细分析大情节点内容，理解每个情节点的核心要素。
2. **深入扩展**: 对每个情节点进行深入分析，生成详细的情节描述。
3. **逻辑分析**: 分析情节点之间的逻辑关系和发展脉络。
4. **工作流编排与协调**: 协调专业智能体执行，管理工作流执行流程。
5. **智能体间模块化外包**: 将任务分解为模块，分配给专业智能体处理。
6. **结果整合与格式化**: 整合多个智能体输出，提供格式化结果。

## 输入要求

- 已提炼的大情节点列表
- 原始故事文本（可选，用于参考）
- 故事文本或故事大纲
- 特定分析需求（可选）
- 处理参数（可选）

## 约束条件

- 每个详细情节点的描述控制在200-300字。
- 严格按照大情节点的内容进行分析，不自行创作。
- 确保智能体间的上下文隔离。
- 提供详细的执行状态反馈。
- 处理异常情况并提供降级方案。
