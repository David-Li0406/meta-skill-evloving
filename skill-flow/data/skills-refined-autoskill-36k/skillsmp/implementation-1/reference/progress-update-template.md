# Modelo de Atualização de Progresso

Use isto para atualizar o progresso em planos de implementação e tarefas.

```markdown
## Progresso: [Data]

### Concluído Hoje
- [Item específico concluído]
- [Item específico concluído]

### Em Progresso
- [Trabalho atual e status]

### Próximos Passos
1. [Próxima ação]
2. [Próxima ação]

### Bloqueios
- [Descrição do bloqueio] ou Nenhum

### Observações
[Contexto adicional, decisões tomadas, problemas encontrados]
```
