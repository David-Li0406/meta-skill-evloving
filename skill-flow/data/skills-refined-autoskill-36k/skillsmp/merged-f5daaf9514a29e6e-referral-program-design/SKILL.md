# Referral Program Designer

Create referral programs that turn happy customers into active promoters. This skill helps you design incentive structures, referral mechanics, and program experiences that motivate sharing and drive cost-effective customer acquisition.

Referrals are the gold standard of marketing—lower CAC, higher LTV, and built-in social proof. This skill provides frameworks for incentive design, program mechanics, viral loop optimization, and measurement. Learn from successful programs to build referral engines that fuel growth.

## Core Workflows

### Workflow 1: Referral Program Design
1. **Goal Setting** - Define what you want the program to achieve.
2. **Customer Motivation Analysis** - Understand why customers would refer.
3. **Incentive Structure** - Determine what to offer referrers and referees.
4. **Reward Mechanics** - Decide when and how rewards are delivered.
5. **Sharing Mechanics** - Establish how referrals are made.
6. **Tracking Design** - Plan for attribution and fraud prevention.
7. **Program Terms** - Outline rules and limitations.

### Workflow 2: Incentive Optimization
1. **Incentive Type Selection** - Choose between cash, credit, product, or status.
2. **Two-Sided Incentives** - Ensure benefits for both parties.
3. **Reward Timing** - Decide between instant or delayed gratification.
4. **Tiered Rewards** - Implement increasing incentives for more referrals.
5. **Milestone Bonuses** - Offer extra rewards for achieving specific goals.
6. **A/B Testing Plan** - Test different incentive structures for effectiveness.
7. **Economics Modeling** - Ensure sustainable unit economics.

### Workflow 3: Program Launch
1. **MVP Program Design** - Create a minimum viable referral program.
2. **Technical Requirements** - Identify needs for tracking, attribution, and rewards.
3. **Communication Strategy** - Plan how to promote the program.
4. **In-Product Integration** - Determine where the referral program fits in the user experience.
5. **Email Sequence** - Develop a nurture sequence for the referral program.
6. **Launch Metrics** - Define success indicators for the launch.
7. **Optimization Roadmap** - Plan for post-launch improvements.

### Workflow 4: Viral Loop Enhancement
1. **Friction Audit** - Identify where users drop off in the referral process.
2. **Share Flow Optimization** - Streamline the sharing process to make it effortless.
3. **Message Customization** - Personalize referral messaging for better engagement.
4. **Multi-Channel Sharing** - Enable sharing through email, social media, and direct messages.
5. **Reminder Strategy** - Re-engage users with referral reminders.
6. **Viral Coefficient Calculation** - Measure the program's virality.
7. **Loop Optimization** - Continuously improve the K-factor over time.

### 5. Viral Coefficient Framework
**K = i × c**
Where:
- i = invitations sent per customer
- c = conversion rate of invitations

| K-Factor | Meaning |
|----------|---------|
| < 0.5 | Weak referrals, needs other channels |
| 0.5-1.0 | Healthy referrals, amplifies growth |
| > 1.0 | Viral growth, self-sustaining |

## Best Practices
- **Make it easy** - Fewer clicks = more referrals.
- **Two-sided incentives** - Benefit both parties.
- **Relevant rewards** - Match incentive to product value.
- **Immediate gratification** - Faster rewards drive behavior.
- **Clear communication** - Simple explanation of how it works.
- **In-product placement** - Referral should be discoverable.
- **Timing matters** - Ask after positive experiences.
- **Personalization** - Customizable referral messages.
- **Social proof** - Show others referring successfully.
- **Multiple channels** - Enable various sharing methods.
- **Fraud prevention** - Design to prevent gaming.
- **Track everything** - Attribution and conversion data.
- **Test constantly** - Optimize incentives and mechanics.
- **Celebrate success** - Recognize top referrers.
- **Sustainable economics** - Ensure CAC savings exceed reward costs.
