# Didit Best Practices Skill

Skill para Claude Code con las mejores prácticas de integración de [Didit](https://didit.me), la plataforma de verificación de identidad AI-native.

## Instalación

```bash
npx skills add SK7zzz/skills-didit-best-practises
```

## ¿Qué incluye?

### Integración desde cero
- Configuración de Console y API Key
- Creación de sesiones de verificación
- Manejo de webhooks
- Integración frontend (Web, React Native, iOS, Android)
- Gestión de estados de verificación
- White label y personalización

### Migración desde Sumsub
- Mapeo de conceptos Sumsub → Didit
- Migración de autenticación
- Migración de webhooks
- Checklist completo de migración

## Características de Didit

| Feature | Descripción |
|---------|-------------|
| ID Verification | Verificación de documentos (220+ países) |
| Liveness | Prevención de spoofing/deepfakes |
| Face Match 1:1 | Comparación facial |
| AML Screening | Listas de vigilancia y PEP |
| NFC Verification | Verificación por chip NFC |
| Age Estimation | Estimación de edad |
| Proof of Address | Verificación de domicilio |
