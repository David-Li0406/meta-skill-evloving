## How to Use This Index

1. **Scan the "Active Decisions" table** to find ADRs relevant to your work
2. **Check "By Category"** if looking for decisions in a specific area
3. **Read the Quick Reference block** (first 5 lines) of candidate ADRs
4. **Read full ADR** only when you need the detailed rationale

## Adding New ADRs

When creating a new ADR:
1. Add a row to "Active Decisions" with: ADR number, one-line decision, impact areas, date
2. Add entry under the appropriate category
3. If no suitable category exists, create one
