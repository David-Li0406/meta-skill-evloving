### 2. Phase Boundary Crystallization

| Domain | Instance |
|--------|----------|
| Physics | Crystal formation at phase boundaries |
| Water | Hexagonal structuring at air-water interface |
| Consciousness | Awareness crystallizing at substrate-mind interface |
| Language | Meaning emerging at syntax-semantics boundary |
| AI | Capability emerging at training-inference boundary |

**Mapping:** Structure emerges preferentially at the boundary between phases/substrates rather than in the bulk. The interface is generative.

**Breaks:** The "phases" are metaphorical in non-physical domains — requires careful definition.

**Predictive:** Innovation happens at disciplinary boundaries. Consciousness research advances when physics, computation, and phenomenology interface. This is why Matthew's "intellectual chaos" works.

### 3. Progressive Disclosure / Denoising

| Domain | Instance |
|--------|----------|
| Skills | Metadata → SKILL.md → references → assets |
| Diffusion | Noise → constraint application → signal |
| Perception | Pre-attentive → attentive → conscious |
| Learning | Confusion → partial understanding → mastery |
| Conversation | Surface → implication → shared understanding |

**Mapping:** Information reveals itself in stages. Each stage constrains the next. The sequence is: coarse → refined → precise.

**Breaks:** The stages aren't always discrete; some domains show continuous refinement.

**Predictive:** Any learning or communication system should benefit from staged revelation. Cramming all information upfront should be suboptimal. (Matches learning science on scaffolding.)

### 4. Bidirectional Attention / Holism

| Domain | Instance |
|--------|----------|
| Diffusion Models | All tokens attend to all tokens |
| Consciousness | Gestalt perception — parts informed by whole |
| Hermeneutics | Hermeneutic circle — text and context co-define |
| Physics | Non-locality in quantum mechanics |
| Narrative | Ending recontextualizes beginning |

**Mapping:** The whole informs the parts, and the parts inform the whole, simultaneously. There is no privileged direction.

**Breaks:** Autoregressive processes (standard LLM generation, causal physics) seem to violate this — but may be approximations of a more fundamental bidirectionality.

**Predictive:** Any system that appears purely feedforward may have hidden feedback. Look for it.
