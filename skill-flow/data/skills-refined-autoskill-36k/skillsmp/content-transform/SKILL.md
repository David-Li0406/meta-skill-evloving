# Content Transform

Transform content between different formats and structures for Showroom labs.

## Supported Transformations

- Asciidoc to Markdown
- Markdown to Asciidoc
- Lab restructuring (reordering sections, splitting/merging modules)
- Template application

## Usage

Specify the source content, target format, and any transformation options.

## Options

- `--format`: Target format (asciidoc, markdown)
- `--template`: Apply a specific template
- `--structure`: Restructure according to a pattern

## Output

Transformed content in the specified format, maintaining semantic meaning and structure where possible.
