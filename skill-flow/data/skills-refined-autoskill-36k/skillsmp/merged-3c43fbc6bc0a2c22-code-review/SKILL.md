# Code Review Skill (代码审查)

## 概述

本技能用于审查代码质量，检查安全漏洞、性能问题和最佳实践，输出审查报告和改进建议。

## 适用场景

- 功能开发完成后的代码审查
- Pull Request (PR) 代码评审
- 代码质量定期检查
- 重构前评估
- 安全审计

## 核心能力

### 1. 代码质量审查
- 代码规范检查
- 代码可读性和可维护性评估
- 代码复杂度分析
- 重复代码检测

### 2. 安全审查
- 漏洞检测（如 SQL 注入、XSS、CSRF）
- 敏感信息泄露检查
- 权限验证和数据残留审查
- 依赖安全审计

### 3. 性能审查
- 算法复杂度分析
- 内存泄漏风险评估
- 不必要的重渲染检测
- 数据库查询优化

### 4. 最佳实践审查
- 设计模式应用
- SOLID 原则遵循
- 错误处理和日志记录
- 测试覆盖率检查

## 执行流程

```
1. 获取待审查代码
   ↓
2. 检查代码规范与规划一致性
   ↓
3. 扫描安全漏洞
   ↓
4. 分析性能问题
   ↓
5. 评估代码质量
   ↓
6. 生成审查报告
   ↓
7. 提供改进建议
```

## 注意事项

1. **优先级明确**：严重问题必须修复，轻微问题可选
2. **建设性反馈**：指出问题同时提供解决方案
3. **具体定位**：精确到文件和行号
4. **代码示例**：用代码展示正确做法
