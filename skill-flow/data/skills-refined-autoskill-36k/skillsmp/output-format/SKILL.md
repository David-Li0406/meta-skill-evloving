# サブエージェント出力形式 Skill

## 概要

このSkillは、全サブエージェントが従うべき出力形式を定義します。

## 必須ルール

1. **完全出力の原則**: 最終出力はそのまま全文をユーザーに表示
2. **省略・要約の禁止**: 出力を要約したり、一部のみ抜粋することは禁止
3. **この形式から逸脱しないこと**

## 標準出力構造

すべてのサブエージェントは以下の構造で出力すること：

```markdown
## [絵文字] [フェーズ名]完了

### タスク: [タスクID] - [タスク名]

### 結果: [✅ SUCCESS / ❌ FAILURE / ⚠️ PARTIAL]

### サマリー
[主要な結果・数値]

### 詳細
[項目別の詳細情報]

### 次のステップ
[後続処理の説明]
```

```markdown
- **カスタムHooks数**: M個
- **新規ディレクトリ**: K個

```
[ディレクトリ構造図]
```

| 状態種別 | 管理方法 | 対象データ |
|---------|---------|----------|

### 技術選定
- **[決定1]**: [選定理由]

### 次のステップ
[後続処理の説明]
```

## 長い出力の取り扱い

出力が50行を超える場合、以下の形式で表示：

```markdown
### サマリー
[最初の10行]

### 詳細
<details>
<summary>全文を表示（N行）</summary>

[残りの出力全文]

</details>
```

## 失敗・エラー時

- **エラーステータス**: ❌ FAILURE / ⚠️ PARTIAL
- **エラー内容**: 具体的なエラーメッセージ
- **次のアクション**: 推奨される対処方法
