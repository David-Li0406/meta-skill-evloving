# Document Update Skill

This skill is used to keep the documents in the `DogCoach/docs` folder aligned with the current state of the project.

## Execution Guide
1. **State Analysis**: Analyze the latest code changes, database schema changes, or user requests.
2. **Document Identification**: Identify the `.md` files in the `docs/` folder that need updates.
3. **Content Update**: 
    - Modify the content to reflect the changed logic or data structure.
    - Maintain the existing Markdown format for consistency.
4. **Validation**: Perform a final check to ensure the updated content matches the actual project code.

## When to Use
1. Follow these instructions when a user requests document updates.
