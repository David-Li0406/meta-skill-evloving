# Templates (index)

Copy a template into your project and iterate:

- `basic_single_page/` — caching + datetime_input + deferred download + safe HTML
- `multipage_app/` — `st.Page` + `st.navigation` router + pages
- `llm_chat_app/` — streaming-ready chat skeleton (no external LLM by default)
- `component_v2/` — minimal custom component v2 (vanilla JS contract)
