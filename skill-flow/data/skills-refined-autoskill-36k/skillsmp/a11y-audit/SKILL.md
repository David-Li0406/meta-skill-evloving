# Чек-лист аудиту доступності

## Критерії перевірки (WCAG 2.2)

1. **Focus Appearance (2.4.13)**: Чи є видимий індикатор фокусу? Чи має він достатній контраст?
2. **Focus Not Obscured (2.4.11, 2.4.12)**: Чи не перекривається елемент у фокусі "липкими" (sticky) елементами? Перевір наявність `scroll-padding-top`.
3. **Target Size (2.5.8)**: Чи мають клікабельні елементи розмір мінімум 24x24 CSS пікселів (краще 44x44)?
4. **Contrast (1.4.3)**: Чи є контраст тексту мінімум 4.5:1 для звичайного тексту?
5. **Labels (3.3.2)**: Чи мають поля форм підписи?
6. **Dragging (2.5.7)**: Якщо є drag-and-drop, чи є альтернатива "в один клік"?
