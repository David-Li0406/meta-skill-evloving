# Production Build

// turbo
1. Build the production bundle:
```bash
cd d:\new\microservices && npm run build
```

2. Verify the build output has no errors

// turbo
3. Start the production server to test:
```bash
cd d:\new\microservices && npm run start
```

4. Open http://localhost:3000 and verify all pages work correctly
