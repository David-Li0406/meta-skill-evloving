# Research Quality Standards

## Core Principles

### Factual Content Only
- Present data with citations, no validation claims
- Include source links and DOIs for academic work
- Avoid subjective assessments
- Focus on verifiable information

### What NOT to Include

**Validation claims:**
- "This data is validated"
- "Research confirms"
- "Studies prove"
- "Evidence shows"

**Subjective assessments:**
- "This is significant"
- "Important findings"
- "Notable results"
- "Impressive data"

**Speculative content:**
- "Could potentially"
- "Might suggest"
- "May indicate"

## Citation Standards

### Academic Papers

```
**Author(s)** (Year). "Title." _Journal_, Volume(Issue), pages.
- DOI: [DOI number]
- Link: [Direct link to paper]
- PubMed: [PubMed link if available]
```

### Industry Reports

```
**Organization** (Year). "Title."
- Link: [Direct link to report]
```

### Data Sources / APIs

```
**Provider** - "Title"
- Cost: Free/Paid
- Coverage: [Scope of data]
- Link: [API or access link]
```

## Quality Checklist

Before completing research:

- [ ] All claims have citations
- [ ] All links are working
- [ ] Data is presented objectively
- [ ] No validation claims included
- [ ] Sources are credible and accessible
- [ ] No duplicate information
