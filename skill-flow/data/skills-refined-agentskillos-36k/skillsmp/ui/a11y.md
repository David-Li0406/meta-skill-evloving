# ♿ 無障礙設計 (Accessibility)

## WCAG 簡介

Web Content Accessibility Guidelines (WCAG) 是國際標準，分為三個等級：

| 等級 | 說明 | 要求程度 |
|-----|------|---------|
| A | 基本 | 最低要求 |
| AA | 中等 | 建議標準（法規常用）|
| AAA | 最高 | 完美標準（較難達成）|

### 四大原則 (POUR)

1. **Perceivable 可感知**：使用者能感知內容
2. **Operable 可操作**：使用者能操作介面
3. **Understandable 可理解**：使用者能理解內容
4. **Robust 穩健**：內容能被各種技術解讀

## 色彩對比

### 對比度要求

| 等級 | 一般文字 | 大文字 | UI 元件 |
|-----|---------|-------|---------|
| AA | 4.5:1 | 3:1 | 3:1 |
| AAA | 7:1 | 4.5:1 | - |

> 大文字：18pt (24px) 或 14pt (19px) 粗體

### 檢測工具

```html
<!-- Chrome DevTools -->
<!-- 1. 開啟開發者工具 -->
<!-- 2. 選取元素 -->
<!-- 3. 在 Styles 面板看對比度 -->
```

### 常見問題與修正

❌ **錯誤**
```css
.placeholder {
  color: #cccccc; /* 對比度不足 */
}
```

✅ **正確**
```css
.placeholder {
  color: #757575; /* 對比度 4.6:1 */
}
```

## 鍵盤導航

### 基本要求

1. 所有互動元素可用 Tab 鍵到達
2. Tab 順序符合視覺順序
3. 當前焦點清楚可見
4. 可用 Enter/Space 觸發操作
5. 可用 Escape 關閉彈窗

### 焦點樣式

```css
/* 不要移除預設焦點樣式 */
/* ❌ 錯誤 */
*:focus {
  outline: none;
}

/* ✅ 正確：自訂但保留焦點指示 */
*:focus {
  outline: none;
}

*:focus-visible {
  outline: 2px solid var(--color-primary);
  outline-offset: 2px;
}
```

### 焦點陷阱 (Focus Trap)

Modal 開啟時，焦點應被限制在 Modal 內：

```javascript
// 基本概念
function trapFocus(element) {
  const focusable = element.querySelectorAll(
    'button, [href], input, select, textarea, [tabindex]:not([tabindex="-1"])'
  );
  const first = focusable[0];
  const last = focusable[focusable.length - 1];

  element.addEventListener('keydown', (e) => {
    if (e.key === 'Tab') {
      if (e.shiftKey && document.activeElement === first) {
        e.preventDefault();
        last.focus();
      } else if (!e.shiftKey && document.activeElement === last) {
        e.preventDefault();
        first.focus();
      }
    }
  });
}
```

## 語意化 HTML

### 使用正確的元素

```html
<!-- ❌ 錯誤 -->
<div onclick="submit()">送出</div>
<div class="link" onclick="navigate()">首頁</div>

<!-- ✅ 正確 -->
<button type="submit">送出</button>
<a href="/">首頁</a>
```

### 結構元素

```html
<header>
  <nav aria-label="主要導航">...</nav>
</header>

<main>
  <article>
    <h1>文章標題</h1>
    <section aria-labelledby="section1">
      <h2 id="section1">段落標題</h2>
    </section>
  </article>
  <aside aria-label="側邊欄">...</aside>
</main>

<footer>...</footer>
```

### 標題結構

```html
<!-- 正確的標題層級 -->
<h1>網站標題</h1>
  <h2>主要區塊</h2>
    <h3>子區塊</h3>
  <h2>另一個區塊</h2>

<!-- ❌ 跳過層級 -->
<h1>標題</h1>
  <h3>錯誤：跳過 h2</h3>
```

## ARIA 屬性

### 常用 ARIA

| 屬性 | 用途 | 範例 |
|-----|------|-----|
| aria-label | 元素標籤 | `<button aria-label="關閉">×</button>` |
| aria-labelledby | 引用標籤 | `<section aria-labelledby="title">` |
| aria-describedby | 額外描述 | `<input aria-describedby="hint">` |
| aria-hidden | 隱藏元素 | `<span aria-hidden="true">裝飾</span>` |
| aria-expanded | 展開狀態 | `<button aria-expanded="false">` |
| aria-current | 當前項目 | `<a aria-current="page">` |
| aria-live | 動態更新 | `<div aria-live="polite">` |

### 範例

```html
<!-- 漢堡選單 -->
<button 
  aria-expanded="false" 
  aria-controls="menu"
  aria-label="開啟選單"
>
  <span class="hamburger-icon" aria-hidden="true"></span>
</button>
<nav id="menu" hidden>...</nav>

<!-- 表單提示 -->
<label for="email">Email</label>
<input 
  id="email" 
  type="email"
  aria-describedby="email-hint email-error"
  aria-invalid="true"
>
<p id="email-hint">請輸入您的電子郵件</p>
<p id="email-error" role="alert">格式不正確</p>
```

## 圖片替代文字

### 原則

| 圖片類型 | alt 處理 |
|---------|---------|
| 資訊性圖片 | 描述內容 |
| 裝飾性圖片 | alt="" |
| 功能性圖片 | 描述功能 |
| 複雜圖片 | 簡短 alt + 詳細描述 |

```html
<!-- 資訊性 -->
<img src="chart.png" alt="2023年銷售成長125%的柱狀圖">

<!-- 裝飾性 -->
<img src="decoration.png" alt="" role="presentation">

<!-- 功能性（按鈕內的圖標）-->
<button>
  <img src="search.svg" alt="搜尋">
</button>

<!-- 複雜圖片 -->
<figure>
  <img src="map.png" alt="公司位置地圖" aria-describedby="map-desc">
  <figcaption id="map-desc">
    公司位於台北市信義區松仁路100號，鄰近捷運象山站2號出口。
  </figcaption>
</figure>
```

## 表單無障礙

### Label 關聯

```html
<!-- 方式一：for 屬性 -->
<label for="name">姓名</label>
<input id="name" type="text">

<!-- 方式二：包裹 -->
<label>
  姓名
  <input type="text">
</label>
```

### 錯誤提示

```html
<label for="email">Email</label>
<input 
  id="email" 
  type="email"
  aria-invalid="true"
  aria-describedby="email-error"
>
<p id="email-error" class="error" role="alert">
  請輸入有效的 Email 格式
</p>
```

### 必填欄位

```html
<label for="name">
  姓名 <span aria-hidden="true">*</span>
  <span class="sr-only">（必填）</span>
</label>
<input id="name" type="text" required aria-required="true">
```

## 動畫與動態

### 減少動態偏好

```css
/* 為動畫設定預設 */
.animated {
  animation: slideIn 0.3s ease;
}

/* 尊重使用者偏好 */
@media (prefers-reduced-motion: reduce) {
  .animated {
    animation: none;
  }
  
  * {
    transition-duration: 0.01ms !important;
  }
}
```

## 工具類

### 視覺隱藏但可讀

```css
/* Screen Reader Only */
.sr-only {
  position: absolute;
  width: 1px;
  height: 1px;
  padding: 0;
  margin: -1px;
  overflow: hidden;
  clip: rect(0, 0, 0, 0);
  white-space: nowrap;
  border: 0;
}

/* 可聚焦時顯示 */
.sr-only-focusable:focus {
  position: static;
  width: auto;
  height: auto;
  overflow: visible;
  clip: auto;
  white-space: normal;
}
```

### 跳過導航

```html
<a href="#main-content" class="skip-link">
  跳到主要內容
</a>

<style>
.skip-link {
  position: absolute;
  top: -100%;
  left: 50%;
  transform: translateX(-50%);
  padding: 1rem;
  background: var(--color-primary);
  color: white;
  z-index: 9999;
}

.skip-link:focus {
  top: 0;
}
</style>
```
