## Typography Rules (STRICTLY ENFORCED)

### 4 Font Sizes ONLY

| Size | Tailwind | Pixels | Use Case |
|------|----------|--------|----------|
| Size 1 | `text-3xl` | 32px | Page headings |
| Size 2 | `text-2xl` | 24px | Section headings |
| Size 3 | `text-base` | 16px | Body text |
| Size 4 | `text-sm` | 14px | Labels, meta |

### 2 Font Weights ONLY

| Weight | Tailwind | Use Case |
|--------|----------|----------|
| Semibold | `font-semibold` | Headings, emphasis |
| Regular | `font-normal` | Body text, UI elements |

### Forbidden

- `text-xs`, `text-lg`, `text-xl`, `text-4xl`, etc.
- `font-light`, `font-medium`, `font-bold`, `font-extrabold`

## 8pt Grid Spacing

All spacing must be divisible by 4 or 8 pixels.

### Allowed Tailwind Spacing

```
p-1 (4px)   m-1 (4px)   gap-1 (4px)
p-2 (8px)   m-2 (8px)   gap-2 (8px)
p-3 (12px)  m-3 (12px)  gap-3 (12px)
p-4 (16px)  m-4 (16px)  gap-4 (16px)
p-5 (20px)  m-5 (20px)  gap-5 (20px)
p-6 (24px)  m-6 (24px)  gap-6 (24px)
p-8 (32px)  m-8 (32px)  gap-8 (32px)
```

### Forbidden

- Custom spacing like `p-[13px]`, `m-[7px]`
- Non-grid values

## 60/30/10 Color Distribution

### 60% - Neutral Backgrounds

```css
bg-background     /* Main page background */
bg-card           /* Card backgrounds */
bg-muted          /* Secondary backgrounds */
```

### 30% - Text & UI

```css
text-foreground        /* Primary text */
text-muted-foreground  /* Secondary text */
border-border          /* Borders */
```

### 10% - Accent ({{BRAND_COLOR}} ONLY)

```css
bg-primary text-primary-foreground  /* CTAs, buttons */
text-primary                        /* Links, emphasis */
ring-primary                        /* Focus rings */
```

### Forbidden

- Hardcoded colors (`bg-blue-500`, `text-gray-700`)
- Multiple competing accent colors
- Overusing accent (>10%)

## Component Standards

### Button Usage

| Variant | Use Case |
|---------|----------|
| `default` | Primary CTAs |
| `secondary` | Secondary actions |
| `destructive` | Dangerous actions |
| `outline` | Tertiary actions |
| `ghost` | Minimal UI |

### Card Spacing

- Header: `p-6`
- Content: `p-6 pt-0`
- Footer: `p-6 pt-0`
- Gap between cards: `gap-4` or `gap-6`

### Form Layout

```svelte
<form class="space-y-4">
  <div class="space-y-2">
    <Label>Field</Label>
    <Input />
  </div>
</form>
```
