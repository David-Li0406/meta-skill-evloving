## ❓ 常见问题

### Q1：这些文档之间有重复吗？

**A**：有意的重复，但角度不同：
- cleanup-strategy.md：完整的"是什么"和"为什么"
- cleanup-integration.md：详细的"如何集成"
- cleanup-operations.md：具体的"怎么做"
- cleanup-quick-ref.md：快速的"查什么"

这样设计是为了让不同角色的用户都能快速找到需要的信息。

---

### Q2：我应该打印哪个文档？

**A**：推荐打印 cleanup-quick-ref.md：
- 包含所有关键信息
- 格式紧凑，易于查询
- 包含打印版清单
- A4纸双面打印即可

---

### Q3：文档多久更新一次？

**A**：
- 清理策略体系稳定，不频繁更新
- 如有重大变更，会更新所有文档
- 建议每季度检查一次是否有更新

---

### Q4：如何反馈文档问题？

**A**：
- 如发现错误或不清楚的地方
- 请记录问题和建议
- 联系项目维护者

---

### Q5：这些文档适用于所有项目吗？

**A**：
- 清理策略体系是通用的
- 但具体的清理层级和阈值可能需要调整
- 建议根据项目规模和特点进行定制
