## Set Mechanics

**Vivid**: Cards care about number of colors among permanents you control. More colors = stronger effects. Rewards 3+ color decks with good mana fixing.

**Hybrid Mana**: Uncommon "Eclipsed" creatures (like Eclipsed Elf, Eclipsed Kithkin) use hybrid mana costs and count as both colors for vivid calculations. These are premium picks.

**Blight**: Black-red mechanic using -1/-1 counters. Creatures with blight place counters on opponents' creatures, triggering "when a creature with a -1/-1 counter dies" effects.

**Changeling**: Creatures with all creature types. Count toward ALL tribal synergies - extremely valuable in this tribal-heavy set.

**Evoke**: Pay alternate cost to cast as instant-speed spell. The creature enters, triggers ETB, then is sacrificed. Great with death triggers.

**Champion**: When creature enters, exile another creature you control of specified type. Returns when champion leaves. Enables re-triggering ETBs.

## Tier List

### S-Tier (First-pickable bombs)
- Ashling's Command - 3-mana board wipe with upside
- Trystan's Command - Flexible removal + card advantage
- Kinbinding - Enchantment that takes over games
- Figure of Fable - Wins games single-handedly
- Gloom Ripper - Recursive threat, hard to answer
- Spinerock Tyrant - 6/6 that dominates combat

### A+ Tier (Premium picks)
- Curious Colossus - Card advantage engine
- Sunderflock - Evasive threat with value
- Ajani, Outland Chaperone - Planeswalker, flexible
- Wren's Masterpiece - Build-around payoff

### A Tier (Strong cards)
- Wistfulness - Card selection and graveyard fill
- Deepchannel Duelist - Unblockable threat
- Loch Mare - Efficient creature with upside
- Eclipsed Elf - Hybrid, counts for vivid, tribal
- Eclipsed Kithkin - Same as above, different colors
- Sanar, Innovative First-Year - Value creature
- Champions of the Shoal - Merfolk payoff
- Ethereal Escort - Strong flyer

### A- Tier (Good cards)
- Grub's Command - Flexible removal
- Chronicle of Victory - Card advantage
- Vibrance - Combat trick with upside
- Aurora Awakener - Elemental synergies
- Moonglove Extract - Artifact removal
- Nameless Inversion - Tribal instant, removal
- Oblivion Ring - Flexible removal

### B+ Tier (Solid playables)
- Most on-color uncommons
- Efficient creatures on curve
- Conditional removal
- Tribal lords

### B Tier (Playable)
- Average commons
- Situational cards
- Filler creatures

### C Tier (Below average)
- Off-tribe creatures
- Overcosted effects
- Narrow sideboard cards

### F Tier (Unplayable)
- Cards that don't work in limited
- Extremely narrow effects

## Ideal Mana Curve (40-card deck)
- 1-drop: 1-2 cards (aggressive decks want 2-3)
- 2-drop: 7-8 cards (most important slot)
- 3-drop: 5-6 cards
- 4-drop: 3-4 cards
- 5-drop: 2-3 cards
- 6+ drop: 0-1 cards
- Lands: 17 (16 for aggressive, 18 for control)

Total creatures: 15-18
Total non-creatures: 5-8

## Build Guidelines

### Sealed Pool Evaluation
1. Sort by color, identify deepest colors
2. Find bombs (S/A+ tier cards)
3. Identify tribal critical mass (4+ of same type)
4. Count removal (want 3-5 pieces)
5. Check mana curve
6. Evaluate mana fixing for splashes

### Draft Priorities
1. Bombs > Premium removal > Efficient creatures
2. Stay open packs 1-2, commit pack 3
3. Tribal payoffs rise in value with more tribal cards
4. Changelings are high picks in any tribal deck
5. Mana fixing (vivid lands) enable powerful splashes

### Common Mistakes
- Forcing a tribe without enough support
- Playing too many 5+ drops
- Not enough 2-drops
- Splashing without fixing
- Ignoring tribal synergies for raw power
