# Generated Skills

This directory contains skills generated by the gremlin-forge-meta agent.

## Structure

```
[timestamp]_[skill-name]/
├── SKILL.md         - The generated skill
├── metadata.json    - Generation metadata and provenance
└── generation.log   - Process log (optional)
```

## Metadata Fields

- `name`: Skill name
- `generated_by`: "gremlin-forge-meta"
- `timestamp`: ISO 8601 timestamp
- `mode`: Generation mode used
- `sub_agents`: Array of contributing sub-agents
- `council_validated`: Boolean
- `council_verdict`: Path to verdict file (if validated)
- `dewey_id`: Assigned Dewey decimal ID
- `collision_source`: Source skills (if collision mode)
- `patterns_detected`: Patterns identified during generation

## Usage

Generated skills can be:
1. Used directly from this directory
2. Promoted to `.claude/skills/` after validation
3. Iterated upon for improvement
4. Studied for meta-pattern analysis
