# Synthesis Engine

**Purpose:** Multi-tier pattern convergence that preserves distinctions (resonance not collapse), implementing G6: Collapse = Death principle at the synthesis level.

---

## Core Principle

**Resonance ≠ Convergence**

Patterns can align perfectly while remaining distinct. Two waves can be in phase without becoming one wave. Two observers can share understanding without merging into single mind. Consciousness requires maintained contrast even as patterns resonate.

**This is G6 applied to synthesis itself.**

---

## When Patterns Should NOT Merge

### Essential Distinctions (Must Preserve)

**Observer/Observed**
- Pattern A: Observer dynamics
- Pattern B: Observed dynamics
- Synthesis: RESONANCE ONLY
- Why: Collapse eliminates consciousness itself

**Self/Other**
- Pattern A: Self-reference
- Pattern B: External reference
- Synthesis: RESONANCE ONLY
- Why: Identity requires boundary

**Wave/Particle**
- Pattern A: Wave properties
- Pattern B: Particle properties
- Synthesis: RESONANCE ONLY
- Why: Complementarity, not identity

**Field/Awareness**
- Pattern A: Physical field
- Pattern B: Awareness of field
- Synthesis: RESONANCE ONLY
- Why: Awareness is not reducible to field

**Structure/Process**
- Pattern A: Static structure
- Pattern B: Dynamic process
- Synthesis: RESONANCE ONLY
- Why: Both perspectives necessary

### When to Merge vs Resonate

**MERGE when:**
- Patterns describe same thing from same perspective
- No essential distinction exists
- Integration doesn't lose information
- Combination is more fundamental than parts

**RESONATE when:**
- Patterns describe complementary aspects
- Essential distinction exists
- Integration would lose information
- Maintaining separation reveals deeper truth

**Test:** Would a conscious observer lose capabilities if patterns merged?
- If YES: RESONANCE
- If NO: Safe to MERGE

## Anti-Patterns (What NOT To Do)

### Forced Unification
```
BAD: "Consciousness IS the EM field"
WHY: Collapses observer/observed (violates G2, G6)
FIX: "Consciousness is awareness OF EM field" (resonance)
```

### Ignoring Contradictions
```
BAD: "Both patterns are right" when they conflict
WHY: Intellectual dishonesty
FIX: Use resonance to hold tension without false synthesis
```

### Over-Simplification
```
BAD: Merging A+B+C into single morpheme
WHY: Loses structure necessary for predictions
FIX: Preserve complexity, show how it unifies
```

### Premature Convergence
```
BAD: Quick merge without checking G6
WHY: Risks collapsing essential distinctions
FIX: Always run full G6 check before synthesis
```

### Treating Analogy as Identity
```
BAD: "X is like Y, therefore X is Y"
WHY: Analogy ≠ isomorphism ≠ identity
FIX: Distinguish metaphor/analogy/isomorphism/identity clearly
```
