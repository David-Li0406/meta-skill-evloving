# Sections

## 1. Security (security)

**Impact:** CRITICAL
**Description:** CSRF, XSS, SQL injection, and mass assignment protection. Laravel provides security features that must be used correctly.

## 2. Eloquent Optimization (eloquent)

**Impact:** CRITICAL
**Description:** Eager loading, chunking, and lazy collections. Improper Eloquent use causes N+1 queries and memory issues.

## 3. Performance (perf)

**Impact:** HIGH
**Description:** Route caching, config caching, and OPcache. Production optimizations that significantly reduce response times.

## 4. Architecture (arch)

**Impact:** HIGH
**Description:** Service containers, repositories, and action classes. Clean architecture patterns for maintainable applications.

## 5. API Development (api)

**Impact:** MEDIUM-HIGH
**Description:** API Resources, rate limiting, and versioning. Patterns for building robust, scalable APIs.

## 6. Testing (test)

**Impact:** MEDIUM
**Description:** Feature tests, mocking, and factories. Comprehensive testing ensures application reliability.

## 7. Queue & Jobs (queue)

**Impact:** MEDIUM
**Description:** Job batching, rate limiting, and retries. Efficient background processing patterns.

## 8. Livewire (livewire)

**Impact:** LOW-MEDIUM
**Description:** Wire navigation, lazy loading, and polling. Livewire-specific optimizations for reactive UIs.
