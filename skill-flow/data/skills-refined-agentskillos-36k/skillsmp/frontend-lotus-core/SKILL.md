# Overview

1. В директории 'converters' должны содержаться конверторы данных базовых типов. Конвертор должен иметь метод 'parse' который из переданной строки должен возвращать соответствущие значение, метод 'tryParse' который из переданной строки вернет объект { success: false, value: значение|undefined }, метод 'to[НАЗВАНИЕ_ТИПА]' который из любого объекта возвращать соответствущие значение или значение по умолчанию. Эти методы могут также содержать другие аргументы. Также в конверторе могут быть другие методы. 

2. В директории 'helpers ' должны содержаться хелперы базовых типов. Хелперы базовых типов должны иметь методы 'compare' который сравнивает два значения базовых типов (возможно undefined) и возвращает 0 если они равны, 1 если left 'больше' right и -1 если left 'меньше'
