# ACT Project Context

## The Harvest

**Focus:** Community regeneration, volunteering, events, therapeutic programs, tenant/vendor management

**Pipelines:**
- Volunteer Pipeline
- Event Booking Pipeline
- Partnership Pipeline
- Tenant/Vendor Pipeline

**Revenue Models:**
- Memberships: $50-200/year
- Event tickets: $20-100
- Therapeutic programs: $80-150/session
- Tenant rent: $500-3000/month

**User Journey:**
Community member → Volunteer → Active member → Program participant → Partner/Tenant

**Special Considerations:**
- Cultural heritage (Kabi Kabi/Jinibara)
- Accessibility
- Sliding scale pricing
