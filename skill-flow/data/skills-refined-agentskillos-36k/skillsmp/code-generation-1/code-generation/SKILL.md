# Code Generation

## 📋 実行前チェック(必須)

### このスキルを使うべきか?
- [ ] ボイラープレートを生成する?
- [ ] スキャフォールディングを行う?
- [ ] テンプレートからコードを生成する?
- [ ] 繰り返しコード構造を自動化する?

### 前提条件
- [ ] プロジェクトのディレクトリ構造を確認したか?
- [ ] 既存の規約・パターンを把握しているか?
- [ ] 生成するコードの目的を理解しているか?

### 禁止事項の確認
- [ ] 生成コードを理解せずにそのまま使おうとしていないか?
- [ ] プロジェクトの規約を無視した構造にしようとしていないか?
- [ ] 不要なコードまで生成しようとしていないか?
