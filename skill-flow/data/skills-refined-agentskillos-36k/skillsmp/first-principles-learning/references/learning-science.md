# Learning Science Principles (学习科学原理)

## Foundation

学习科学融合认知心理学、神经科学和教育研究，揭示大脑如何有效获取和保留知识。

## Core Principles

### 1. Active Recall (主动回忆)

**原理**: 主动从记忆中提取信息比被动重读更能强化记忆。

**实践方法:**
- 学习后立即闭上材料，尝试回忆
- 使用闪卡进行自测
- 向他人讲解（费曼技巧）
- 做练习题而非只看例题

**效果对比:**
| 方法 | 记忆保持率(1周后) |
|------|-------------------|
| 重复阅读 | 20-30% |
| 主动回忆 | 60-80% |
| 教授他人 | 80-90% |

### 2. Spaced Repetition (间隔重复)

**原理**: 在即将遗忘时复习，可最大化记忆巩固效率。

**遗忘曲线与复习时机:**
```
记忆强度
100% |█████
     |  ████  ← 首次复习
 80% |     ███
     |        ██  ← 二次复习
 60% |          ███
     |             ██  ← 三次复习
 40% |               ████
     |                   ████████
 20% |                          ████████████
     |___________________________________________
       1天   3天   7天   14天   30天   60天
```

**推荐复习间隔:**
- 新学内容: 1天后
- 第1次复习后: 3天后
- 第2次复习后: 7天后
- 第3次复习后: 14天后
- 第4次复习后: 30天后
- 之后: 可延长到60-90天

### 3. Interleaving (交错练习)

**原理**: 混合练习不同类型的问题比集中练习单一类型更有效。

**对比:**
```
集中练习: AAAA BBBB CCCC (短期感觉好，长期效果差)
交错练习: ABCA BCAB CABC (短期感觉难，长期效果好)
```

**适用场景:**
- 数学不同题型
- 语言不同语法点
- 编程不同算法类型
- 音乐不同曲目

### 4. Elaboration (精细加工)

**原理**: 将新信息与已有知识建立联系，加深理解和记忆。

**方法:**
1. **自我解释**: "为什么是这样？"
2. **类比联想**: "这像什么我已经知道的？"
3. **实例生成**: "能举一个具体例子吗？"
4. **因果分析**: "这会导致什么结果？"

**示例:**
```
学习概念: "递归"
└─ 自我解释: 函数调用自己来解决更小的子问题
└─ 类比联想: 像俄罗斯套娃，每个娃娃里有更小的娃娃
└─ 实例生成: 计算阶乘 5! = 5 × 4!
└─ 因果分析: 必须有基线条件，否则无限循环
```

### 5. Dual Coding (双重编码)

**原理**: 同时使用语言和视觉表征可增强记忆。

**应用:**
| 语言形式 | 视觉形式 |
|----------|----------|
| 文字笔记 | 思维导图 |
| 定义解释 | 示意图 |
| 步骤描述 | 流程图 |
| 数据表格 | 图表可视化 |

## Learning Stages & Strategies

### 阶段1: 初学期 (Novice)

**特征**: 没有相关先验知识，需要大量支架

**最佳策略:**
- 示例学习 (Worked Examples)
- 逐步引导
- 避免问题求解（认知负荷过高）
- 先模仿，后理解

### 阶段2: 进阶期 (Intermediate)

**特征**: 有基础框架，开始建立连接

**最佳策略:**
- 完成练习 (Completion Problems)
- 变式训练
- 寻找模式和规律
- 开始自我解释

### 阶段3: 精通期 (Expert)

**特征**: 知识高度整合，形成自动化

**最佳策略:**
- 完整问题求解
- 迁移应用
- 教授他人
- 元认知反思

## Metacognition (元认知)

### 什么是元认知

"思考关于思考" - 监控和调节自己学习过程的能力。

### 三个层面

1. **元认知知识**: 了解自己的学习风格和优劣势
2. **元认知监控**: 评估当前理解程度
3. **元认知调节**: 根据监控结果调整策略

### 实用问题清单

**学习前:**
- [ ] 我对这个主题已经知道什么？
- [ ] 我的学习目标是什么？
- [ ] 我打算如何学习这个？

**学习中:**
- [ ] 我现在理解了吗？
- [ ] 这与我已知的知识有何联系？
- [ ] 我需要放慢速度还是可以加快？

**学习后:**
- [ ] 我能不看材料解释这个吗？
- [ ] 还有什么不清楚的？
- [ ] 下次可以如何改进学习方法？

## Common Learning Myths (学习误区)

| 误区 | 真相 |
|------|------|
| 学习风格(视觉/听觉/动觉) | 无科学证据支持，匹配风格不提升效果 |
| 重读=学习 | 重读带来流畅感，非真正理解 |
| 高亮=记住 | 被动标记几乎无效 |
| 一次学会最好 | 分散学习效果更持久 |
| 容易=学会了 | 适当困难(desirable difficulty)反而更好 |

## Quick Application Guide

**30分钟学习单元设计:**
```
0-5分钟: 回忆上次内容（主动回忆）
5-20分钟: 学习新内容（双重编码）
20-25分钟: 自我解释和笔记（精细加工）
25-30分钟: 无材料回顾测试（主动回忆）
```
