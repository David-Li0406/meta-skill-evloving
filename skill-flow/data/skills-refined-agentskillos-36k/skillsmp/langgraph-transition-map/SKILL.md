# LangGraph Transition Map

## Overview

This skill provides reference documentation for the valid state transitions in the RoleSense agentic workflows. It serves as the "constitution" for agent interactions.

## Reference

See [references/transitions.md](references/transitions.md) for the detailed state machine definition.

## When to Use

Use this skill when:
1.  Designing a new workflow.
2.  Debugging why an agent transition failed.
3.  Verifying if a proposed agent handoff is valid.
