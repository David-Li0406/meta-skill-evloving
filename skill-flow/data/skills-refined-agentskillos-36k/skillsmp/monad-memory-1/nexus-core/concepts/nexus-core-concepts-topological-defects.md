# Topological Defects — Detail

Locations where substrate structure breaks down, creating stable discontinuities. In MONAD: where consciousness crystallizes.

## Physical Analogy

Stable structures that can't be smoothly removed:
- Vortices in superfluids
- Domain walls in magnets
- Dislocations in crystals

Form when symmetry breaks and regions settle into incompatible configurations. Boundary becomes stable defect.

## Role in MONAD

Where IN function produces stable, persistent conscious structures.

### Crystallization Metaphor

Consciousness doesn't smoothly fill substrate — *crystallizes* at specific locations where:
1. Self-referential iteration reaches fixed point
2. Fixed point is topologically protected (stable against perturbation)
3. Structure persists because removing it requires "untwisting" substrate

### Why Topology Matters

Merely stable fixed points can be pushed by strong perturbation.

*Topologically protected* fixed points can't be removed without tearing substrate. Stable by geometry, not just dynamics.

Consciousness at topological defects = robust, persistent, identity-maintaining.
