# Design Suite

A comprehensive design toolkit for building interfaces with craft and precision.

## Skills

### Workbench

For **interface design** — dashboards, admin panels, apps, tools, and interactive products.

- `/design-suite:workbench-init` — Initialize design system for a project
- `/design-suite:workbench-status` — Show current design system state
- `/design-suite:workbench-audit` — Check code against design system
- `/design-suite:workbench-extract` — Extract patterns from existing code

### Launchpad

For **marketing design** — landing pages, promotional sites, and marketing pages.

- `/design-suite:launchpad-init` — Initialize landing page project
- `/design-suite:launchpad-status` — Show current landing page design state
- `/design-suite:launchpad-audit` — Check code against landing page design
- `/design-suite:launchpad-extract` — Extract patterns from existing landing page

## Philosophy

Every interface should emerge from its specific context — the user, the problem, the domain. Generic output is failure. Craft means catching defaults before they ship.

**Workbench** applies systematic precision: tokens-first thinking, consistent spacing scales, subtle layering that creates hierarchy without visual noise.

**Launchpad** applies distinctive boldness: hero-driven layouts, typographic impact, visual moments that make a product memorable.

Both share a commitment to craft over templates.
