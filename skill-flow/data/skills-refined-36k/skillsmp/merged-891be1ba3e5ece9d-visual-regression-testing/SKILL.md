# Body of the merged SKILL.md

This skill encompasses two main functions related to visual regression testing:

## Check Visual Test Status
- Check the status of visual regression tests, including results, failures, and summaries. This function is applicable in both local TDD and cloud modes.

## Debug Visual Regression
- Analyze and debug visual regression test failures, including investigating screenshot differences and visual bugs. This function is also applicable in both local TDD and cloud modes.
