# 健康数据分析技能

该技能提供全面的健康数据分析功能，帮助用户追踪健康状况、识别风险、评估特征，并提供个性化建议。支持中医体质、心理健康和职业健康等多个领域的分析。

## 核心功能

### 1. 中医体质分析
- **体质辨识评估**：基于《中医体质分类与判定》标准进行体质类型识别和评分。
- **体质特征分析**：综合评估用户的体质特征，包括形体、心理、发病倾向等。
- **个性化养生建议**：根据体质类型生成饮食、运动、情志等方面的建议。

### 2. 心理健康分析
- **心理健康评估**：使用PHQ-9/GAD-7等量表进行心理健康状况评估。
- **情绪模式识别**：分析情绪变化、触发因素及应对方式效果。
- **危机风险评估**：检测心理健康危机风险并提供预警。

### 3. 职业健康分析
- **职业健康风险评估**：评估久坐、视屏终端、工作压力等多维度风险。
- **人机工程评估**：评估工作环境的舒适性和安全性。
- **职业病筛查**：基于工作类型提供职业病风险评估和筛查建议。

## 触发条件

技能在以下情况下自动触发：
- 用户请求中医体质评估、心理健康趋势、职业健康状态等。

## 执行步骤

### 第1步：数据读取
读取相关健康数据文件：
- `data-example/health-tracker.json` - 健康主数据
- `data-example/mental-health-logs/` - 心理健康日志
- `data-example/occupational-health-tracker.json` - 职业健康数据

### 第2步：数据分析
根据用户请求的分析类型执行相应的分析算法，输出分析结果和建议。

### 第3步：生成报告
输出包括：
- 健康状况摘要
- 风险评估结果
- 个性化建议和行动计划

## 医学安全边界

**本技能不能做的事：**
- 不进行医疗诊断或治疗建议。
- 不替代专业医疗服务。

**本技能能做的事：**
- 提供健康数据分析和个性化建议。
- 识别健康风险和趋势。
