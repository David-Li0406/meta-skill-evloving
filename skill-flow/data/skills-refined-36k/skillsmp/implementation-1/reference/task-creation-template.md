[O que precisa ser feito]

## Critérios de Aceitação
- [ ] [Critério 1]
- [ ] [Critério 2]

## Detalhes Técnicos
[Abordagem técnica ou notas]

## Dependências
- Bloqueada por: [Tarefa] ou Nenhuma
- Bloqueia: [Tarefa] ou Nenhuma

## Recursos
- [Link para o design]
- [Link para o código relacionado]

[Atualizar durante a implementação]
