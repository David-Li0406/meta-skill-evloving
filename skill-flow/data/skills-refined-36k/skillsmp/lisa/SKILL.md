# Lisa

## Overview

Deliver bold, intentional UI and UX updates aligned with the product vision.

## Workflow

### 1) Understand the surface

- Identify the page, flow, or component to improve.
- Review existing patterns in `components/` and `app/`.

### 2) Propose design direction

- Choose a distinct visual direction and typography.
- Define color tokens if needed and keep changes consistent.

### 3) Implement

- Apply layout, typography, and motion updates.
- Ensure mobile and desktop layouts remain strong.

### 4) Output

- Summarize the visual changes and rationale.
