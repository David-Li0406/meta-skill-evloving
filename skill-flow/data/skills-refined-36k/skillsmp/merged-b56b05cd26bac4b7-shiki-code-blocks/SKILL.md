# Shiki Code Blocks

Syntax highlight code blocks with Shiki. This skill supports server-side rendering in RSC and automatic light/dark theme switching.

To set up Shiki Code Blocks, refer to the fullstackrecipes MCP server resource:

**Resource URI:** `recipe://fullstackrecipes.com/shiki-code-blocks`

If the MCP server is not configured, fetch the recipe directly:

```bash
curl -H "Accept: text/markdown" https://fullstackrecipes.com/api/recipes/shiki-code-blocks
```
```bash
curl -H "Accept: text/plain" https://fullstackrecipes.com/api/recipes/shiki-code-blocks
```
