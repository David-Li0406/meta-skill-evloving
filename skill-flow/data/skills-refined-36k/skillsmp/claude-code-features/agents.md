# Agents (Sub-agents)

**Trigger**: Manual invocation or delegation from Claude

## Characteristics

- **Independent context** (doesn't pollute main conversation)
- Parallel execution possible
- Can have specialized personas

## Best For

- Code reviewer
- Test runner
- Security auditor
- Large-scale code analysis
- Debugger

## Not Suitable For

- Processing requiring parent context (→ Skills)
- Simple standard tasks (→ Commands)
