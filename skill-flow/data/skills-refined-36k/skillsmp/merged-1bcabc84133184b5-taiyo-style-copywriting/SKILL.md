# Taiyo Style Copywriting Skill

## Overview

This skill combines the generation of Taiyo-style headlines and postscript (P.S.) messages, designed to significantly enhance open rates, click-through rates, and conversion rates in marketing communications.

## When to Use

```
「太陽スタイルでヘッドラインを作って」
「セールスレターの追伸を作って」
「キャッチコピーを10案考えて」
「メルマガの件名を最適化して」
「/taiyo-style-copywriting」
```

## ヘッドライン生成

### 12の黄金パターン

1. **数字+ベネフィット型**
   ```
   【公式】[数字] + [期間] + [ベネフィット]
   ```
   **効果**: 具体性UP、信憑性UP

2. **疑問+好奇心型**
   ```
   【公式】なぜ[逆説的事実]なのか？
   ```
   **効果**: 好奇心喚起、続きを読ませる

3. **警告+恐怖型**
   ```
   【公式】【警告] [ターゲット]が知らない[リスク]
   ```
   **効果**: 危機感喚起、行動促進

4. **Before→After型**
   ```
   【公式】[どん底の状態] → [成功した状態]
   ```
   **効果**: 変化の可能性を示す

5. **秘密暴露型**
   ```
   【公式】[権威者]が絶対に教えない[秘密]
   ```
   **効果**: 独占欲、特別感

6. **限定+緊急型**
   ```
   【公式】【期限】+ [限定条件] + [特典/機会]
   ```
   **効果**: 即決促進、FOMO喚起

7. **反常識型**
   ```
   【公式】[常識の否定] という衝撃の事実
   ```
   **効果**: 意外性、知的好奇心

8. **ターゲット特定型**
   ```
   【公式】[具体的なターゲット]限定
   ```
   **効果**: 自分事化、除外による価値向上

9. **ストーリー予告型**
   ```
   【公式】[衝撃的な結末]までの物語
   ```
   **効果**: 感情移入、続きへの期待

10. **証拠提示型**
    ```
    【公式】[具体的な証拠] を公開
    ```
    **効果**: 信頼性、具体性

11. **比較優位型**
    ```
    【公式】[競合/常識] vs [自分の方法]
    ```
    **効果**: 差別化、選択の明確化

12. **約束型**
    ```
    【公式】この[コンテンツ]で[具体的な成果]を約束
    ```
    **効果**: 期待感、コミットメント

## 追伸生成

### 7つの黄金パターン

1. **緊急性強調型**
   ```
   P.S. 締切は◯月◯日23:59です。残り◯時間で、この機会は完全に消えます。
   ```

2. **損失回避型**
   ```
   P.S. 正直に言います。この手紙を読んで何もしない人が、おそらく90%以上でしょう。
   ```

3. **希望提示型**
   ```
   P.S. 3ヶ月後のあなたを想像してみてください。毎朝、目覚まし時計なしで起きる。
   ```

4. **保証強調型**
   ```
   P.S. まだ迷っていますか？大丈夫です。30日間の完全返金保証があります。
   ```

5. **特典追加型**
   ```
   P.S. 実は、まだお伝えしていないことがあります。今日中にお申し込みいただいた方には、特別に「◯◯◯」をプレゼントします。
   ```

6. **社会的証明型**
   ```
   P.S. この手紙を公開してから、すでに◯◯名の方がお申し込みくださっています。
   ```

7. **個人的メッセージ型**
   ```
   P.S. 最後に、個人的なことを言わせてください。私はあなたの味方です。一緒に乗り越えましょう。
   ```

## 品質チェックリスト

### ヘッドライン
- [ ] 具体的な数字が含まれている
- [ ] ターゲットが明確
- [ ] ベネフィットが伝わる
- [ ] 20-50文字以内（用途による）
- [ ] 感情を動かすワードがある

### 追伸
- [ ] 締切または限定性が明記
- [ ] 本文を読まなくても理解できる
- [ ] CTAリンクまたは行動指示がある
- [ ] 感情的な締めくくり
