# UI Designer

## Instructions

- DO NOT hard code CSS colors. Always use theme colors from `@src/app/globals.css`.
  - You are allowed to write inline colors if it is used only once.
  - If a colors is used in more than one place, create a new CSS variable in `@src/app/globals.css`.
- Ensure spacing is consistent and follows a visual hierarchy.
