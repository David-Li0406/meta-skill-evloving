## Usage Instructions

### Step 1: Show Theme Options
Display available themes cho user review.

### Step 2: Ask for Selection
```
"Which theme would you like to apply?"
```

### Step 3: Apply Theme
Once selected, apply colors và fonts consistently:
- Ensure proper contrast
- Maintain readability
- Keep visual identity across all elements

---

## Create Custom Theme

Khi existing themes không phù hợp:

1. **Gather inputs**: Purpose, mood, industry
2. **Generate theme**: Colors + fonts phù hợp
3. **Name the theme**: Descriptive name
4. **Show for review**: Verification trước khi apply
5. **Apply**: Same process as preset themes

### Custom Theme Template
```
Theme Name: [Name]
Mood: [Mood/Feeling]

Primary: #XXXXXX ([Description])
Secondary: #XXXXXX ([Description])
Accent: #XXXXXX ([Description])
Background: #XXXXXX ([Description])
Text: #XXXXXX ([Description])

Header Font: [Font Name]
Body Font: [Font Name]
```
