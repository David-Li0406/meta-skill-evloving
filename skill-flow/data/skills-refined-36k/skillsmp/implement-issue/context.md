# Gathering Context

Use these tools to understand the issue:

- `mcp__github__get_issue` - Retrieve issue details
- `mcp__github__get_issue_comments` - Get comments on the issue
- `mcp__github__search_issues` - Find related issues (only if alluded to but not linked)
- `mcp__github__search_code` - Find related code (only if identifiers appear to reference code)

For Linear or GitLab issues, load the respective skill for service-specific tools.
