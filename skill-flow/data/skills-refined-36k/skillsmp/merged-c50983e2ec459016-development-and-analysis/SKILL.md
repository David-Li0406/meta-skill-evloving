# 開発と分析スキルパッケージ

このスキルパッケージは、要件の実装、コード分析、及び跨境電商運営に必要な一連の機能を提供します。

## 実装サイクル

要件を分割し、開発エージェントとレビューエージェントを交互に使用して、高品質な実装を完成させます。

### フェーズ1: 要件分析とスコープ分割

1. ユーザーの要件を分析する
2. 要件を適切なスコープ（実装単位）に分割する
3. 各スコープの依存関係を特定し、実装順序を決定する
4. タスクリストを作成する

### フェーズ2: 実装サイクル（各スコープに対して繰り返し）

各スコープに対して以下を実行：

#### ステップA: プランニング
- スコープの要件を明確にする
- 影響を受けるファイルとコンポーネントを特定
- 既存のコードベースのパターンを確認

#### ステップB: 開発フェーズ
- `develop` エージェントを起動
- 現在のスコープの要件を明確に伝える
- 最小限の変更で要件を満たす実装を行う
- テストも含めて実装する

#### ステップC: レビューフェーズ
- `review` エージェントを起動
- 実装されたコードをレビューする
- 問題点、改善点を特定する

#### ステップD: 改善サイクル
- レビューで指摘された問題がある場合：
  - 再度 `develop` エージェントで修正を実施
  - 再度 `review` エージェントでレビュー
  - 指摘事項がなくなるまで繰り返す
- 指摘事項がなくなったら、次のスコープへ進む

### フェーズ3: 完了確認

1. すべてのスコープの実装が完了したことを確認
2. 全体のテストを実行して動作確認
3. 実装内容のサマリーをユーザーに報告

## コード分析・移行スキル

このスキルは、コードベースの分析と移行に必要な機能を提供します。

### 含まれるスキル

1. **RepoConnector（リポジトリ接続）**
   - Git/SVNリポジトリに接続し、コードベース情報を取得します。

2. **StaticAnalyzer（静的分析）**
   - コードの品質問題、バグ、セキュリティ問題を検出します。

3. **ComplexityScorer（複雑度評価）**
   - コードの複雑度を評価し、リファクタリング優先度を提案します。

4. **DependencyMapper（依存関係マッピング）**
   - 依存関係を分析し、依存グラフを構築します。

5. **SecurityScanner（セキュリティスキャン）**
   - セキュリティ脆弱性を検出します。

6. **MigrationPlanner（移行計画策定）**
   - レガシーコードの移行計画を策定します。

## 跨境電商運営スキル

このスキルは、跨境電商運営に必要な一連の作業を自動化します。

### 含まれるスキル

1. **CompetitorScraper（競品採集）**
   - ECプラットフォームから競合商品の情報を収集します。

2. **PriceAnalyzer（価格分析）**
   - 競合商品の価格を分析し、最適な価格戦略を提案します。

3. **ListingGenerator（Listing生成）**
   - SEO最適化された商品リスティングを生成します。

4. **InventoryAdjuster（在庫・価格調整）**
   - 市場分析に基づいて在庫と価格を調整します。

5. **AdMonitor（広告監視）**
   - 広告キャンペーンのパフォーマンスを監視します。

6. **DailyReportGenerator（日報生成）**
   - 毎日の運営データを集約してレポートを生成します。

## ワークフロー

```
要件分析 → 実装サイクル → コード分析 → 跨境電商運営
    ↓            ↓              ↓              ↓
  スコープ分割   開発・レビュー   分析・移行   商品管理
```

## 注意事項

- 各スコープは独立して実装・テスト可能な単位にする
- スクレイピングは各プラットフォームの利用規約に従ってください
- 実際の商用利用前に法的確認を行ってください
