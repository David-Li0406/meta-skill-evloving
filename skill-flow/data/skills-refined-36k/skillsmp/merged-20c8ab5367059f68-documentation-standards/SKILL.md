# Body of the merged SKILL.md

## JavaScript Documentation (JSDoc)

- **Purpose**: Use this section when writing JSDoc comments for JavaScript functions, classes, or modules.
- **When to Use**: Documenting JavaScript code, generating API docs with JSDoc, or improving existing JS documentation.
- **When Not to Use**: For TypeScript code (use TSDoc), Python (use python-docstring), PHP (use php-docstring), or runtime logic without documentation needs.

## TypeScript Documentation (TSDoc)

- **Purpose**: Use this section when writing TSDoc comments for TypeScript functions, classes, interfaces, or types.
- **When to Use**: Documenting TypeScript code, generating docs with TypeDoc, or improving existing TS documentation.
- **When Not to Use**: For JavaScript code (use JSDoc), Python (use python-docstring), PHP (use php-docstring), or runtime logic without documentation needs.
