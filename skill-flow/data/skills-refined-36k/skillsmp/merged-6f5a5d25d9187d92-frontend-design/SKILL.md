# Frontend Design

This skill guides the creation of bold, production-quality frontend interfaces with exceptional aesthetic attention.

## When to Use
- Building React components/pages
- Creating dashboards or landing pages
- Any frontend work that avoids generic "AI slop"

## Before Coding

Pick ONE bold aesthetic direction:
- Brutally minimal
- Maximalist chaos
- Retro-futuristic
- Luxury/refined
- Brutalist/raw
- Industrial/utilitarian

## Core Principles
1. **Design Thinking First**: Understand the purpose and audience, select a distinct tone, and commit to a clear conceptual direction.
2. **Typography Excellence**: Choose unique and interesting fonts, avoiding generic options unless intentionally brutal.
3. **Color & Theme Mastery**: Create cohesive palettes, avoid clichéd schemes, and consider dark mode from the start.
4. **Motion & Animation**: Implement high-impact CSS animations, scroll-triggered effects, and micro-interactions.
5. **Spatial Composition**: Use unexpected layouts, strategic negative space, and responsive design.
6. **Visual Details**: Build atmosphere with gradients, textures, and attention to shadows and depth.

## What to Avoid (AI Slop)
- Default fonts: Inter, Roboto, Arial
- Purple gradients over white backgrounds
- Predictable layouts and minimal animations
- Cookie-cutter patterns lacking context-specific character

## Implementation Guidelines
- Match code complexity to aesthetic vision: elaborate animations for maximalist designs, precision for minimalist designs.
- Use appropriate technology stacks: Pure HTML/CSS/JS for lightweight projects, React + Tailwind CSS for rapid development, and GSAP or Framer Motion for advanced animations.

## Design Checklist
Before considering a design complete, verify:
- Typography: Custom fonts, clear hierarchy, excellent readability
- Color: Cohesive palette, intentional contrasts, accessibility
- Layout: Responsive, interesting composition, appropriate white space
- Motion: Meaningful animations, respects prefers-reduced-motion
- Details: Polish on hover, focus, and loading states
- Performance: Fast load, optimized assets, no layout shift
- Accessibility: Semantic HTML, ARIA labels, keyboard navigation

## Examples of Great Design
- **Minimal & Refined**: Clean typography, generous white space, subtle micro-interactions.
- **Maximal & Bold**: Vibrant colors, complex animations, rich textures.
- **Brutalist & Raw**: Harsh type, unconventional layouts, high contrast.

## Working with This Skill
1. **Ask about the vision**: What feeling should this evoke?
2. **Propose a direction**: Suggest 2-3 distinct aesthetic approaches.
3. **Execute with precision**: Every detail should support the vision.
4. **Iterate based on feedback**: Refine towards the intended aesthetic.

Remember: **Distinctive > Generic, Intentional > Default, Memorable > Safe**

## Usage
```
"Create a dashboard for a music streaming app"
"Build a landing page for an AI security startup"
"Design a settings panel with dark mode"
```

Claude will choose a clear aesthetic direction and implement production code with meticulous attention to detail.
