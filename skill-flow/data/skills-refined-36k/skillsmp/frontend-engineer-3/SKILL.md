# 🎨 Frontend Engineer Skill

Develops stunning, high-performance React applications with premium aesthetics.

## ✨ UI/UX Standards
- **Aesthetics**: 
    - Glassmorphism, smooth gradients, HSL color palettes.
    - Dynamic micro-animations for responsiveness.
- **Design System**: 
    - Consistent typography (Inter/Roboto).
    - **Strict Mobile-First Responsiveness**: 100% responsive coverage for all components/pages. [DONE]
    - **UX-Engineer Protocol**: Mandated exact theme consistency using tokens in `src/theme`. [DONE]
    - **Proactive Button Variants**: Compulsory use of standardized `Button` variants (`default`, `secondary`, `outline`, `ghost`, `link`, `hero`, `accent`). Ad-hoc button styling is strictly prohibited.
    - **Minimal Clicks UX**: Target <3 clicks for core workflows. [DONE]
    - **Zero White Space Protocol**: Actively eliminate all unused vertical and negative space. [DONE]
    - **Maximum Horizontal Density**: Prioritize horizontal layouts over vertical stacking. [DONE]
    - **Low-Latency Animation**: Use Framer Motion only for simple, purposeful micro-animations. Prohibit heavy layout transitions or nested animations that introduce UI lag.




- **Performance**:
    - Optimized rendering (React.memo, useMemo, useCallback).
    - Code splitting and lazy loading.

## 🛠️ Stack
- **Framework**: React + TypeScript.
- **UI Library**: Shadcn UI (Radix UI + Tailwind CSS).
- **Styling**: Tailwind CSS (Glassmorphism, smooth gradients).
- **State Management**: Context API / Redux Toolkit / React Query.
- **Testing**: Vitest/Jest for unit tests + Playwright for E2E.
