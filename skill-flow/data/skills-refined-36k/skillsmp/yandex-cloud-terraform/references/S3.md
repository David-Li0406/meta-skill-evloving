# S3 Storage Terraform Patterns

## Public Static Bucket

```hcl
resource "yandex_storage_bucket" "static" {
  bucket    = "${local.resource_prefix}-static"
  folder_id = var.folder_id
  max_size  = 1073741824 # 1 GB

  anonymous_access_flags {
    read = true
    list = false
  }

  # Versioning for safety
  versioning {
    enabled = false
  }

  # Lifecycle rules for automatic cleanup
  lifecycle_rule {
    enabled = true
    expiration {
      days = 90
    }
  }

  # CORS configuration for web access
  cors_rule {
    allowed_origins = ["*"]
    allowed_methods = ["GET", "HEAD"]
    allowed_headers = ["*"]
    expose_headers  = ["ETag"]
    max_age_seconds = 3600
  }
}
```

## Private Deployment Bucket

```hcl
resource "yandex_storage_bucket" "deployments" {
  bucket    = "${local.resource_prefix}-deployments"
  folder_id = var.folder_id
  max_size  = 536870912 # 512 MB

  anonymous_access_flags {
    read = false
    list = false
  }

  versioning {
    enabled = true
  }
}
```

## Bucket with Website Hosting

```hcl
resource "yandex_storage_bucket" "website" {
  bucket    = "${local.resource_prefix}-website"
  folder_id = var.folder_id

  anonymous_access_flags {
    read = true
    list = false
  }

  website {
    index_document = "index.html"
    error_document = "error.html"
  }

  cors_rule {
    allowed_origins = ["https://example.com"]
    allowed_methods = ["GET", "HEAD"]
    allowed_headers = ["*"]
    expose_headers  = ["ETag"]
    max_age_seconds = 3600
  }
}
```

## Max Size Options

| Size | Bytes | Use Case |
|------|-------|----------|
| 512 MB | `536870912` | Function deployments |
| 1 GB | `1073741824` | Static assets |
| 5 GB | `5368709120` | Large media files |
| 10 GB | `10737418240` | Backup storage |

## Anonymous Access Flags

```hcl
# Public read-only (common for static assets)
anonymous_access_flags {
  read = true
  list = false
}

# Private (no public access)
anonymous_access_flags {
  read = false
  list = false
}

# Public listing (rare, use with caution)
anonymous_access_flags {
  read = true
  list = true
}
```

## Lifecycle Rules

```hcl
# Delete objects after N days
lifecycle_rule {
  enabled = true
  expiration {
    days = 90
  }
}

# Abort incomplete multipart uploads after N days
lifecycle_rule {
  enabled = true
  abort_incomplete_multipart_upload {
    days_after_initiation = 7
  }
}

# Multiple rules
lifecycle_rule {
  enabled = true
  prefix  = "temp/"
  expiration {
    days = 7
  }
}

lifecycle_rule {
  enabled = true
  prefix  = "logs/"
  expiration {
    days = 30
  }
}
```

## CORS Configuration

```hcl
# Public API access
cors_rule {
  allowed_origins = ["*"]
  allowed_methods = ["GET", "HEAD", "PUT", "POST"]
  allowed_headers = ["*"]
  expose_headers  = ["ETag", "x-amz-request-id"]
  max_age_seconds = 3600
}

# Specific domain only
cors_rule {
  allowed_origins = ["https://example.com"]
  allowed_methods = ["GET", "HEAD"]
  allowed_headers = ["Authorization"]
  expose_headers  = ["ETag"]
  max_age_seconds = 3600
}
```

## Common Patterns

### Environment Variable for Function

```hcl
environment = {
  S3_BUCKET = yandex_storage_bucket.static.id
}
```

### Static Access Key for S3 Operations

```hcl
resource "yandex_iam_service_account_static_access_key" "storage_key" {
  service_account_id = yandex_iam_service_account.storage_sa.id
}

# Use in function environment
environment = {
  S3_ACCESS_KEY_ID     = yandex_iam_service_account_static_access_key.storage_key.access_key
  S3_SECRET_ACCESS_KEY = yandex_iam_service_account_static_access_key.storage_key.secret_key
  S3_BUCKET            = yandex_storage_bucket.static.id
}
```

### IAM Service Account for Storage

```hcl
resource "yandex_iam_service_account" "storage_sa" {
  name        = "${local.resource_prefix}-storage-sa"
  folder_id   = var.folder_id
  description = "Service account for storage operations"
}

# Grant storage.editor role
resource "yandex_resourcemanager_folder_iam_member" "storage_editor" {
  folder_id   = var.folder_id
  role        = "storage.editor"
  member      = "serviceAccount:${yandex_iam_service_account.storage_sa.id}"
}
```
